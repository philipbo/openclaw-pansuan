# 联赛配置手册（泊松 + 盘口）

> 目标：同一套泊松与盘口方法，在不同联赛下用**不同参数和基准**来套，从而「说人话」。

---

## 一、为什么要按联赛配置？

泊松公式本身无需变，但：

- 各联赛的 **平均进球数** 不同；
- **平局 / 小比分倾向** 不同；
- **主场优势** 强弱不同；
- **盘口习惯**（让球 / 大小球）也不同。

如果不按联赛区分这些基准：

- 同样总进球 2.6，在荷乙只是中性偏小，在意乙却已经是明显偏大的队；
- 同样半一盘，在英超强队 vs 中游是常规，在意甲就可能是偏深的名气盘。

因此：

> **模型只有一套，参数按联赛分开。**  
> λ 公式、泊松、EV 公式都一样，只是用不同联赛的「平均总进球」「盘位深浅标准」「风格标签」去喂。

---

## 二、每个联赛需要配置哪些东西？

每个联赛至少准备一张配置表，包含：

1. **联赛平均总进球 G_league**  
   - 示例（仅为示意，需用实际数据更新）：
     - 英超：≈ 2.8–3.0  
     - 西甲：≈ 2.5  
     - 意甲：≈ 2.4  
     - 德甲：≈ 3.0  
     - 法甲：≈ 2.5  
     - 荷乙：≈ 3.1  
     - 意乙 / 法乙：≈ 2.2–2.3
2. **典型大小球线分布**  
   - 该联赛常见的大 / 小盘口位置：
     - 荷乙：2.75 / 3 为常规；
     - 意乙：2 / 2.25 / 2.5 为常规；
     - 英超：2.5 / 2.75 为常规。  
   - 用于判断某场「2.5 盘」在该联赛里是偏大还是偏小。
3. **平局 / 小比分倾向**  
   - 高 / 中 / 低 三档：
     - 某些赛季的意甲 / 法甲 → 平局 / 小比分偏多；
     - 德甲 / 荷甲 → 分胜负和大比分偏多。  
   - 影响比分权重中对 0:0、1:1、1:0 等比分的基础看法。
4. **主场优势强弱**  
   - 主场优势强 / 中 / 弱：
     - 土超、部分南美联赛 → 主场强；
     - 部分五大联赛近年主场优势减弱。  
   - 影响你看「让球盘深浅」时的容忍度。
5. **盘口风格备注**  
   - 如：
     - 英超强队让半一、一球较常见；
     - 意甲让深盘要提高警惕名气盘；
     - 荷乙大球盘（3/3.25）非常常见。

## 三、如何在现有手册中使用联赛配置

你已有的手册：

- `泊松概率计算手册.md`
- `盘口与走势阅读手册.md`
- `比分建模与投注手册.md`
- `球队风格判定手册.md`
- `下注决策手册.md`

在这些手册里涉及到「联赛平均」「大球 / 小球联赛」「盘深 / 盘浅」的地方，应按联赛查表，而不是用固定数字。

### 3.1 在 λ 与风格判定中的使用

- 在 `球队风格判定手册.md` 中：
  - 判定「总进球偏大 / 偏小」时，用 **G_league** 作为分界：
    - 总进球 ≥ G_league+0.3 → 偏大球队；
    - 总进球 ≤ G_league−0.3 → 偏小球队；
  - 判定「进攻强 / 防守强」时，进失球也相对 G_league 而不是绝对值。  
- 在计算 λ 时，对「近 6 场」与「赛季总场均」的权重也可轻微按联赛调整：
  - 荷乙波动大，可略提高近 6 场权重；
  - 意甲结构稳，可略偏赛季场均。

### 3.2 在比分权重与大小球中的使用

- 在 `比分建模与投注手册.md` 中：
  - 对类型 A/B/C（偏大 / 中性 / 偏小）队伍的划分，使用各自联赛的 G_league 和典型大小盘位置；
  - 对 0:0、1:1 等小比分是否给更高 w(i:j)，按「该联赛平局 / 小比分倾向」微调。  
- 大小球方向判断时：
  - 同样 λ 下，大 / 小 2.5 在荷乙和意乙的含义不同，需要用「典型大小球线」来校准直觉。

### 3.3 在盘深 / 盘浅判断中的使用

- 在 `盘口与走势阅读手册.md` 中：
  - 平半、半球、半一、一球这类让球盘的「常规深浅」标准，按联赛配置：
    - 英超强 vs 中游：半一 / 一球可视为常规强队盘；
    - 意甲强 vs 中游：同样盘位要更警惕名气盘；
  - 大 / 小盘的深浅同理：
    - 荷乙 3 球不算非常夸张；
    - 意乙 3 球几乎就是极端大球预期。

## 四、简单示例：同样总进球 2.6 在不同联赛的含义

### 4.1 荷乙

- 假设荷乙 G_league≈3.0；
- 某队总进球（得+失）=2.6：
  - 略低于联赛平均 → 在本联赛中偏向「中性略偏小」；
  - 大小盘常开 2.75 / 3，对这队来说 2.5 算偏小盘。

### 4.2 意乙

- 假设意乙 G_league≈2.2；
- 某队总进球=2.6：
  - 明显高于联赛平均 → 在本联赛中是偏大的队；
  - 大小盘多在 2 / 2.25 时，这队的 2.5 盘可能已经偏大。

**同一个 2.6，总进球在荷乙与意乙里的风格标签完全不同，比分和大小球的权重处理也应不同。**

## 五、落地建议

1. 为你常下注的联赛（英超、意甲、西甲、德甲、法甲、荷甲、荷乙、意乙等）各写一小节，填上：
   - 估计的 G_league；
   - 典型大小球线位置；
   - 平局 / 小比分倾向；
   - 主场优势强 / 中 / 弱；
   - 一两条盘口风格备注。
2. 以后在任何手册里遇到「联赛平均」等字样时，按本文件查对应表，而不是用死数字。  
3. 定期（比如每半赛季）用实际数据更新 G_league 和联赛特征，修正你的直觉。

> 长期来看，**模型的骨架可以不变，联赛配置的表要经常更新**。你的下注质量，很多时候取决于这些「看似是常识、实际是参数」的小东西是否跟得上时代。EOF
