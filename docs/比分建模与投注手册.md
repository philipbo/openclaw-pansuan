# 比分建模与投注手册（泊松版）

> 目标：只围绕「比分」这一种玩法，整理一套你能反复使用的建模与选注方法。

---

## 一、从 λ 到比分概率

前置：

- 已算出本场的 λ₁（主队预期进球）、λ₂（客队预期进球）；
- 已按泊松公式算出：
  - 主队进 0～4 球的概率 P(主=0…4)；
  - 客队进 0～4 球的概率 P(客=0…4)。

### 1.1 单场比分概率 P(i:j)

对每个比分 (i, j)：

- **P(主 i : 客 j) = P(主=i) × P(客=j)**（假设主客进球数独立）。
- 通常只算到 0～4 球（甚至 0～5），其它归为「尾部」。

### 1.2 胜平负与比分的关系

- 主胜概率 = ∑ P(i,j) 其中 i>j；
- 平局概率 = ∑ P(i,j) 其中 i=j；
- 客胜概率 = ∑ P(i,j) 其中 i<j。

比分表既可以单独用来玩正确比分，也可以作为「底层」支撑胜平负、让球盘、大 / 小球的判断。

## 二、如何从比分表中选「主攻比分」

### 2.1 基本思路

对单场比分玩法，直接追「最可能的那个比分」命中率不会很高（一般最高的比分也就 8%～12%）。更合理的思路是：

1. 先按概率从大到小排一个比分列表；
2. 结合盘口（胜平负、让球、大 / 小）和足球本质（风格、战意、伤停）做筛选；
3. 最后选出 2～4 个「主攻比分」，而不是硬梭一个。

### 2.2 操作步骤

1. **从泊松文档里抄出前 N 个比分**：
   - 比如前 8～12 个，累计概率通常能覆盖 70% 左右。
2. **按胜平负方向先划掉一批**：
   - 如果你综合判断更看好「主不败」，可以先剔除客胜类比分（如 0:1、1:2 等），少量保留用于防冷即可；
   - 若盘面和模型都指向客强主弱，则相反处理。
3. **用大小球信息再过滤**：
   - 如果大小球 2.5 明显偏小（泊松 + 盘口都支持小球），优先保留 0:0、1:0、1:1、2:0 这类小比分；
   - 若偏大球，则更看重 2:1、2:2、3:1、1:2 等。
4. **用基本面做最后筛选**：
   - 强队在「必须赢 / 需净胜」场景中，大胜比分（3:0、3:1、4:1）权重可以略微上调；
   - 两队都偏保守、进攻疲软，更多倾向 0:0、1:0、1:1、0:1 一类的小比分。

> 实战建议：
>
> - 正确比分永远是低命中高赔率玩法，不建议大量梭单一比分；
> - 更适合「少量资金、多场组合」或作为你主玩法（胜平负 / 亚盘）的辅助加注。

## 三、用泊松 + 赔率筛选「有价值的比分」

比分玩法的 EV 和胜平负完全一样：

- **EV(比分 i:j) = P(i:j) × 该比分的赔率**。

步骤：

1. 取前 8～12 个高概率比分 P(i:j)；
2. 对应每个比分的市场赔率 O(i:j)（尽量用欧指平均或主流书）；
3. 计算 EV(i:j) = P(i:j) × O(i:j)；
4. **只考虑 EV≥1.05 的比分作为价值候选**，其余当作「方向参考」而不下注。（EV 处于 1.00～1.05 的边际区间，默认记笔记但不强推）

> 注意：
>
> - 很多比分的市场赔率已经内含高抽水，哪怕 P 不低，EV 也可能 <1；
> - 真正 EV 明显>1 的比分不会很多，遇到时也要结合盘口、基本面确认不是模型误差。

## 四、如何把比分和胜平负 / 让球盘结合

### 4.1 从胜平负 / 亚盘反推比分范围

1. 若你已经选定某个方向（例如客胜或客+0.5）作为主玩法：
   - 优先考虑与该方向一致的比分（1:2、0:1、1:3…），
   - 不必强求覆盖所有方向。
2. 若你主玩法是「主不败」：
   - 优先考虑 1:0、2:1、1:1、2:0、0:0 这类与主不败兼容的比分。

### 4.2 用比分辅助「盘口纠结」

- 当胜平负或让球盘方向模糊（比如主胜、平概率接近）：
  - 看比分分布里平局类比分（0:0、1:1、2:2）的合计占比；
  - 若平局比分累积占比明显高于某一方胜出比分，可以适度提高「防平」权重，或在比分玩法中增加平局比分比重。

## 五、提升比分命中率的几点实战建议

1. **优先从总进球数分布入手**：
   - 先看 P(0球)、P(1球)、P(2球)… 哪一档最高；
   - 再在该总进球数对应的比分里选最合理的几个（比如总进球=2 时，1:1、2:0、0:2 三个）。
2. **关注球队风格和极端场景**：
   - 防守极强的队（主打 1:0、0:0）、进攻/防守都极端的队（经常打 3:2、2:3）；
   - 决赛、次回合、保级生死战等特殊场景，比分分布会明显偏离常规泊松（比如最后阶段疯狂进攻或死守）。
3. **不要强行覆盖太多比分**：
   - 2～4 个主攻比分 + 少量防守比分（比如多一个 1:1 或 2:1）更合理；
   - 比赛越多时，比分组合更适合作为小额高赔率「博花」玩法，而不是主仓位。
4. **长期统计自己的比分预测质量**：
   - 每场记录：
     - 泊松下的前 N 名比分列表；
     - 实际比分是否在列表中、排第几；
   - 逐步调试 λ 的计算方式和筛选逻辑，让「真实比分落在你主攻区间内」的频率稳步提高。

---

## 六、一句话策略

> **比分玩法的核心，不是猜「哪一个比分必中」，而是：
> 1）用泊松明确大概率比分区间；
> 2）用盘口、大小球、基本面筛掉明显不合理的比分；
> 3）在赔率合适（EV≥1.05）的前提下，主攻 2～4 个与你主玩法方向一致的比分，长期拉高命中率和整体收益。**
