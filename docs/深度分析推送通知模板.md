# 深度分析推送通知模板

供 agent 严格按此结构推送单场分析报告；龙王可据此检查是否缺项、是否按模板执行。

---

## 一、推送结构总览

**必须先按模板完成全部 10 步分析**，再组稿推送。若总字符数 **超过 4000**（飞书等渠道有长度限制），须分段发送，不得为省字数省略必含块或步骤。

- **分段方式**：在「━━━ 分析过程 ━━━」与「━━━ 结论 ━━━」之间拆成 2 条（第 1 条＝分析过程，第 2 条＝结论）。
- **衔接**：首条末尾加「⬇️ 接下文」，续条开头加「⬆️ 接上文」。
- **每段开头**（可选，便于飞书等折叠）：`【深度分析 {编号} 第 N 段/共 M 段】`

---

## 二、必含块与顺序

### 1. 标题与场次信息（首段开头）

```
【深度分析 {编号} 第 1 段/共 M 段】
📊 深度分析 [{联赛}] {主队} VS {客队}
🆔 比赛 ID: {matchId} | 竞彩编号：{编号}
⏰ 开球：{YYYY-MM-DD HH:mm} | 🏟️ {球场} | 天气：{可选}
```

### 2. 分析过程（第 1～3 段，可多段）

- **第 1 步：基本面**（排名、近况、主客场、交锋、伤停等）
- **第 2 步：伤停**（双方缺阵与影响）
- **自开盘**（欧指/亚盘/大小球合理区间）
- **第 3 步：欧指分析**
  - **欧指赔率变动轨迹分析（必做）**：
    - 时点对齐表（初盘 / T1 时间 / T2 时间 / … / 即时盘，各公司主/平/客）。**时点格式**：初盘/即时盘不标时间；T1、T2、… 为共同时点，时间用 `MM-DD HH:mm`（如 T1 03-15 12:30）或同场简写 `HH:mm`（如 T1 12:30），与步骤 1.5 抓取的「变化时间」一致。
    - 每家意图小结（除平博）
    - 相同时点交叉认证
    - 一句话解读
  - 六层排查小结
- **第 4 步：亚盘分析**
  - **亚盘赔率变动轨迹分析（必做）**：
    - 时点对齐表（初盘 / T1 / T2 / … / 即时盘，各公司 盘口/主水/客水）；时点含义与格式同第 3 步。
    - 每家意图小结（除平博）
    - 相同时点交叉认证
    - 一句话解读
  - 六层排查小结
- **第 5 步：大小球**
  - **大小球赔率变动轨迹分析（必做）**：
    - 时点对齐表（时点格式同第 3 步）、每家意图、相同时点认证、一句话解读
  - 小结：**先写大 X 球/小 X 球**（如大 2.5/3 球、小 2/2.5 球），再写预期总进球（避免只写「预期进球 2-3 球」而不交代方向）
- **第 6～10 步摘要**（赔率合理性、关键点/疑点/矛盾、泊松 λ、Top 3 比分等）

若此处总长超 4000 字，在「结论」前拆条，并加 ⬇️ 接下文 / ⬆️ 接上文。

### 3. 结论（最后一段，必须含以下全部）

```

🎯 综合评估与推荐

编号：{编号}（无则 -） | 比赛ID：{matchId} | 开球：{开球时间}
🕐 分析时间：{YYYY-MM-DD HH:mm}
🎯 推荐：
  ⚽ 胜平负：{主胜/平局/客胜}（或主结论 + 备选）
    └ 赔率依据：{…}
  🏈 让球：{…}（盘口：{X}）｜或 ⚠️ 观望（原因）
    └ 赔率依据：{…}
  📏 大小球：{大X.5/小X.5}（盘口：{X.5}，预期总进球：{λ1+λ2}）｜或 ⚠️ 观望
    └ 赔率依据：{…}
  🔢 比分（按概率排序）：
    1️⃣ {比分} ({概率}%)
    2️⃣ {比分} ({概率}%)
    3️⃣ {比分} ({概率}%)

📊 置信区间：主胜 {XX-XX%} / 平局 {XX-XX%} / 客胜 {XX-XX%}
🔑 关键点：…
❓ 疑点：…
⚔️ 核心矛盾：…
💪 信心度：{高/中/低}
🔗 串关适合度：{🔒做胆 / ✅可串 / ⚠️慎串}
⚠️ 风险：…
```
