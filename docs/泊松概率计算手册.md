# 泊松概率计算手册（足彩用）

本文档总结从基础概念到完整计算流程的泊松模型用法，便于复现与实现。

---

## 一、泊松基础

### 1.1 公式

在固定时间（一场 90 分钟比赛）内，某队进球数 **X** 服从泊松分布时：

$$
P(X = k) = \frac{\lambda^k \cdot e^{-\lambda}}{k!}
$$

- **k**：进球数（0, 1, 2, 3, …）
- **λ**：该队在本场的**预期进球数**（唯一参数）
- **e**：自然对数底 ≈ 2.718
- **k!**：k 的阶乘（0!=1, 1!=1, 2!=2, 3!=6, …）

### 1.2 性质

- 期望与方差：**E(X) = λ**，**Var(X) = λ**
- 主队进球 X₁ ~ Poisson(λ₁)，客队进球 X₂ ~ Poisson(λ₂)，且**独立**时：
  - 比分 (i, j) 概率：**P(主 i : 客 j) = P(X₁=i) × P(X₂=j)**
  - 总进球期望：**E(X₁ + X₂) = λ₁ + λ₂**

---

## 二、足彩中的输出项

给定 **λ₁（主队）** 和 **λ₂（客队）** 后，可得到：

| 输出       | 含义                    | 计算方式                              |
| ---------- | ----------------------- | ------------------------------------- |
| 主队进球数 | 主队进 0/1/2/… 球的概率 | P(主=k) = λ₁^k·e^{-λ₁}/k!             |
| 客队进球数 | 客队进 0/1/2/… 球的概率 | P(客=k) = λ₂^k·e^{-λ₂}/k!             |
| 预期总进球 | 全场预期总进球          | λ₁ + λ₂                               |
| 比分       | 具体比分 (i,j) 的概率   | P(主=i)×P(客=j)                       |
| 胜平负     | 主胜/平/客胜概率        | 对 i>j / i=j / i<j 的 P(i,j) 分别求和 |
| 总进球数   | 总进球为 n 的概率       | 对所有 i+j=n 的 P(i,j) 求和           |
| 大 2.5 球  | 总进球 ≥3               | P(3)+P(4)+…                           |
| 小 2.5 球  | 总进球 ≤2               | P(0)+P(1)+P(2)                        |

---

## 三、λ 的计算（核心流程）

### 3.1 所需数据

每队需要三类统计（建议至少主/客各 5～6 场再单独用主客场数据）：

| 维度                           | 含义                       | 最少建议场次 |
| ------------------------------ | -------------------------- | ------------ |
| **总**                         | 联赛全场得失球             | 10+ 场       |
| **主**（主队）/ **客**（客队） | 主队主场、客队客场的得失球 | **5～6 场**  |
| **近6**                        | 最近 6 场得失球（状态）    | 4～6 场      |

从表格中提取：

- 主队：**主场**场均进球、**主场**场均失球；**近6** 场均进球、场均失球
- 客队：**客场**场均进球、**客场**场均失球；**近6** 场均进球、场均失球

### 3.2 步骤一：基础 λ（仅用主队主场 + 客队客场）

公式：

$$
\lambda_1 = \frac{\text{主队主场场均进} + \text{客队客场场均失}}{2}
$$

$$
\lambda_2 = \frac{\text{客队客场场均进} + \text{主队主场场均失}}{2}
$$

- **λ₁**：主队本场预期进球
- **λ₂**：客队本场预期进球

主队主场场均进/失 = 主队主场总进球/总失球 ÷ 主队主战场次；客队客场同理。

### 3.3 步骤二：掺入近六场（加权）

先算「合成攻防」再代入上面公式，权重建议 **0.7 主客场 + 0.3 近6**：

**主队**

- 合成进攻 = 0.7 × 主队主场场均进 + 0.3 × 主队近6 场均进
- 合成防守 = 0.7 × 主队主场场均失 + 0.3 × 主队近6 场均失

**客队**

- 合成进攻 = 0.7 × 客队客场场均进 + 0.3 × 客队近6 场均进
- 合成防守 = 0.7 × 客队客场场均失 + 0.3 × 客队近6 场均失

再用合成值算 λ：

$$
\lambda_1 = \frac{\text{主队合成进攻} + \text{客队合成防守}}{2}, \quad
\lambda_2 = \frac{\text{客队合成进攻} + \text{主队合成防守}}{2}
$$

得到**合成 λ**（未伤停前）。

### 3.4 步骤三：伤停修正（可选）

在合成 λ 上直接加减，**不改变公式结构**，只改 λ₁、λ₂ 的数值：

| 情况                          | 建议调整（幅度可依信息细调）          |
| ----------------------------- | ------------------------------------- |
| 主队进攻主力缺阵              | λ₁ 减 0.1～0.2                        |
| 主队后防大面积伤              | λ₂ 加 0.1～0.15                       |
| 客队进攻主力缺阵              | λ₂ 减 0.1～0.2                        |
| 客队后防大面积伤              | λ₁ 加 0.1～0.15                       |
| 伤情身份不明（主力/替补未知） | 用较小幅度（如 ±0.05～0.1），或暂不调 |

最终：

- **λ₁** = 合成 λ₁ + 主队相关修正
- **λ₂** = 合成 λ₂ + 客队相关修正
- **预期总进球** = λ₁ + λ₂

---

## 四、从 λ 到各项概率的计算步骤

### 4.1 主队 / 客队进球数分布

对 k = 0, 1, 2, 3, 4, 5, 6, … 计算：

$$
P(\text{主}=k) = \frac{\lambda_1^k \cdot e^{-\lambda_1}}{k!}, \quad
P(\text{客}=k) = \frac{\lambda_2^k \cdot e^{-\lambda_2}}{k!}
$$

一般算到 k=5 或 6 即可，剩余归入「5+」或「6+」。可用递推：  
P(0)=e^{-λ}，P(k)=P(k-1)×λ/k。

### 4.2 比分概率

$$
P(\text{主 }i : \text{客 }j) = P(\text{主}=i) \times P(\text{客}=j)
$$

主客各取 k=0～5（或 0～6），做成矩阵，逐格相乘即得各比分概率。

### 4.3 胜平负

- **主胜** = ∑ P(i,j)，对所有 **i > j**
- **平** = ∑ P(i,j)，对所有 **i = j**
- **客胜** = ∑ P(i,j)，对所有 **i < j**

三者之和为 1。

### 4.4 总进球数概率

总进球 **n** = i + j：

$$
P(\text{总进球}=n) = \sum_{i+j=n} P(\text{主}=i) \times P(\text{客}=j)
$$

对 n=0,1,2,… 分别求和。例如 n=2 时，(i,j)=(2,0),(1,1),(0,2) 三种。

### 4.5 大 / 小 2.5 球

- **大 2.5 球** = P(总进球≥3) = P(3)+P(4)+P(5)+…
- **小 2.5 球** = P(总进球≤2) = P(0)+P(1)+P(2)

---

## 五、完整计算清单（按顺序执行）

1. **整理数据**：主队主场的得/失、近6 得/失；客队客场的得/失、近6 得/失 → 算出各场均。
2. **基础 λ**：用「3.2」公式，仅主队主场 + 客队客场，算 λ₁、λ₂。
3. **合成攻防**：用「3.3」的 0.7/0.3 加权，得到主客各的合成进攻、合成防守。
4. **合成 λ**：用合成攻防再代入「3.2」中的 λ 公式，得到合成 λ₁、λ₂。
5. **伤停修正**：按「3.4」对 λ₁、λ₂ 加减，得到**最终 λ₁、λ₂**。
6. **预期总进球**：λ₁ + λ₂。
7. **泊松分布**：用最终 λ₁、λ₂ 算 P(主=k)、P(客=k)，k=0～5 或 6。
8. **比分**：P(i,j)=P(主=i)×P(客=j)，列成矩阵。
9. **胜平负**：按 i>j / i=j / i<j 对 P(i,j) 分类求和。
10. **总进球**：按 n=i+j 对 P(i,j) 分组求和，得 P(总进球=n)。
11. **大/小 2.5**：P(≥3) 与 P(≤2)。

---

## 六、样本与主客场说明

- **主队主场、客队客场**：建议至少 **5～6 场** 再单独用于算 λ；不足时可混入「总」或降低主客场权重。
- **近 6 场**：固定取最近 6 场（不足 6 场则有几场算几场，可适当降低近6 权重）。
- 基础 λ 若**已用主队主场 + 客队客场**，则主客场因素已包含，一般**不再**额外乘「主场优势系数」，避免重复。

---

## 七、公式速查

| 项目         | 公式                                  |
| ------------ | ------------------------------------- |
| 单队进球概率 | P(X=k) = λ^k·e^{-λ}/k!                |
| 基础 λ₁      | (主队主场场均进 + 客队客场场均失) / 2 |
| 基础 λ₂      | (客队客场场均进 + 主队主场场均失) / 2 |
| 合成攻防     | 0.7×主/客场均 + 0.3×近6场均           |
| 比分         | P(i:j) = P(主=i)×P(客=j)              |
| 预期总进球   | λ₁ + λ₂                               |
| 主胜/平/客胜 | ∑P(i,j) 其中 i>j / i=j / i<j          |
| 总进球=n     | ∑\_{i+j=n} P(主=i)×P(客=j)            |
| 大 2.5       | P(总进球≥3)                           |

以上即为从数据到胜平负、预期总进球、总进球数、比分的完整公式与步骤总结。
