# 泊松概率计算手册（足彩用）

本文档总结从基础概念到完整计算流程的泊松模型用法，便于复现与实现。

---

## 一、泊松基础

### 1.1 公式

在固定时间（一场 90 分钟比赛）内，某队进球数 **X** 服从泊松分布时：

$$
P(X = k) = \frac{\lambda^k \cdot e^{-\lambda}}{k!}
$$

- **k**：进球数（0, 1, 2, 3, …）
- **λ**：该队在本场的**预期进球数**（唯一参数）
- **e**：自然对数底 ≈ 2.718
- **k!**：k 的阶乘（0!=1, 1!=1, 2!=2, 3!=6, …）

### 1.2 性质

- 期望与方差：**E(X) = λ**，**Var(X) = λ**
- 主队进球 X₁ ~ Poisson(λ₁)，客队进球 X₂ ~ Poisson(λ₂)，且**独立**时：
  - 比分 (i, j) 概率：**P(主 i : 客 j) = P(X₁=i) × P(X₂=j)**
  - 总进球期望：**E(X₁ + X₂) = λ₁ + λ₂**

---

## 二、足彩中的输出项

给定 **λ₁（主队）** 和 **λ₂（客队）** 后，可得到：

| 输出       | 含义                    | 计算方式                              |
| ---------- | ----------------------- | ------------------------------------- |
| 主队进球数 | 主队进 0/1/2/… 球的概率 | P(主=k) = λ₁^k·e^{-λ₁}/k!             |
| 客队进球数 | 客队进 0/1/2/… 球的概率 | P(客=k) = λ₂^k·e^{-λ₂}/k!             |
| 预期总进球 | 全场预期总进球          | λ₁ + λ₂                               |
| 比分       | 具体比分 (i,j) 的概率   | P(主=i)×P(客=j)                       |
| 胜平负     | 主胜/平/客胜概率        | 对 i>j / i=j / i<j 的 P(i,j) 分别求和 |
| 总进球数   | 总进球为 n 的概率       | 对所有 i+j=n 的 P(i,j) 求和           |
| 大 2.5 球  | 总进球 ≥3               | P(3)+P(4)+…                           |
| 小 2.5 球  | 总进球 ≤2               | P(0)+P(1)+P(2)                        |

---

## 三、λ 的计算（核心流程）

### 3.1 所需数据

每队需要三类统计（建议至少主/客各 5～6 场再单独用主客场数据）：

| 维度                           | 含义                       | 最少建议场次 |
| ------------------------------ | -------------------------- | ------------ |
| **总**                         | 联赛全场得失球             | 10+ 场       |
| **主**（主队）/ **客**（客队） | 主队主场、客队客场的得失球 | **5～6 场**  |
| **近6**                        | 最近 6 场得失球（状态）    | 4～6 场      |

从表格中提取：

- 主队：**主场**场均进球、**主场**场均失球；**近6** 场均进球、场均失球
- 客队：**客场**场均进球、**客场**场均失球；**近6** 场均进球、场均失球

### 3.2 步骤一：基础 λ（仅用主队主场 + 客队客场）

公式：

$$
\lambda_1 = \frac{\text{主队主场场均进} + \text{客队客场场均失}}{2}
$$

$$
\lambda_2 = \frac{\text{客队客场场均进} + \text{主队主场场均失}}{2}
$$

- **λ₁**：主队本场预期进球
- **λ₂**：客队本场预期进球

主队主场场均进/失 = 主队主场总进球/总失球 ÷ 主队主战场次；客队客场同理。

### 3.3 步骤二：掺入近六场（加权）

先算「合成攻防」再代入上面公式，权重建议 **0.7 主客场 + 0.3 近6**：

**主队**

- 合成进攻 = 0.7 × 主队主场场均进 + 0.3 × 主队近6 场均进
- 合成防守 = 0.7 × 主队主场场均失 + 0.3 × 主队近6 场均失

**客队**

- 合成进攻 = 0.7 × 客队客场场均进 + 0.3 × 客队近6 场均进
- 合成防守 = 0.7 × 客队客场场均失 + 0.3 × 客队近6 场均失

再用合成值算 λ：

$$
\lambda_1 = \frac{\text{主队合成进攻} + \text{客队合成防守}}{2}, \quad
\lambda_2 = \frac{\text{客队合成进攻} + \text{主队合成防守}}{2}
$$

得到**合成 λ**（未伤停前）。

### 3.4 步骤三：伤停修正（可选）

在合成 λ 上直接加减，**不改变公式结构**，只改 λ₁、λ₂ 的数值：

| 情况                          | 建议调整（幅度可依信息细调）          |
| ----------------------------- | ------------------------------------- |
| 主队进攻主力缺阵              | λ₁ 减 0.1～0.2                        |
| 主队后防大面积伤              | λ₂ 加 0.1～0.15                       |
| 客队进攻主力缺阵              | λ₂ 减 0.1～0.2                        |
| 客队后防大面积伤              | λ₁ 加 0.1～0.15                       |
| 伤情身份不明（主力/替补未知） | 用较小幅度（如 ±0.05～0.1），或暂不调 |

最终：

- **λ₁** = 合成 λ₁ + 主队相关修正
- **λ₂** = 合成 λ₂ + 客队相关修正
- **预期总进球** = λ₁ + λ₂

### 3.5 伤停修正幅度建议（实战版）

伤停信息不完整时，不要用健康版 λ 直接算价值。建议按伤停严重程度分三档处理：

- **轻微伤停**：两队各 0～2 人，且以轮换/替补为主 → **不修正 λ**
- **中度伤停**：某队 3～4 人伤，含 1～2 个主力 → 可做小幅修正（±0.05～0.10）
- **重度伤停**：
  - 某队 **≥5 人伤**，且有多名主力（尤其是门将、中卫、中锋）；或
  - 双方伤病人数相差 ≥3（例如 9 vs 2）  
    → **必须修正 λ**，否则模型不可信；若不想修，整场标记为「不做价值下注」。

简化实战规则（不区分攻守 λ，直接作用到 λ₁、λ₂）：

- **前场主力受损**（边锋/中锋等）
  - 主力前场伤 1 人：本队 λ 减 **0.05～0.10**
  - 主力前场伤 2 人：本队 λ 减 **0.10～0.20**
  - 主力前场伤 ≥3 人：本队 λ 减 **0.20～0.30**，并标记为高风险场次
- **后防主力受损**（门将、中卫、边后卫）
  - 后防主力伤 1 人：对手 λ 加 **0.05～0.10**
  - 后防主力伤 2 人：对手 λ 加 **0.10～0.20**
  - 后防主力伤 ≥3 人：对手 λ 加 **0.20～0.30**，并标记为高风险场次
- **又伤前锋又伤后卫**：本队 λ 同时减，对手 λ 同时加（幅度取上述范围高值），同时在笔记中标记为「极端伤停场次」。

> 实战建议：
>
> - 若出现「重度伤停」且你不确定如何修正 λ，宁可将本场标记为「模型不适用，不做价值下注」。
> - 只有在明确按上述规则修正 λ 后，才继续后面的泊松计算与赔率比较。

## 四、从 λ 到各项概率的计算步骤

### 4.1 主队 / 客队进球数分布

对 k = 0, 1, 2, 3, 4, 5, 6, … 计算：

$$
P(\text{主}=k) = \frac{\lambda_1^k \cdot e^{-\lambda_1}}{k!}, \quad
P(\text{客}=k) = \frac{\lambda_2^k \cdot e^{-\lambda_2}}{k!}
$$

一般算到 k=5 或 6 即可，剩余归入「5+」或「6+」。可用递推：  
P(0)=e^{-λ}，P(k)=P(k-1)×λ/k。

### 4.2 比分概率

$$
P(\text{主 }i : \text{客 }j) = P(\text{主}=i) \times P(\text{客}=j)
$$

主客各取 k=0～5（或 0～6），做成矩阵，逐格相乘即得各比分概率。

### 4.3 胜平负

- **主胜** = ∑ P(i,j)，对所有 **i > j**
- **平** = ∑ P(i,j)，对所有 **i = j**
- **客胜** = ∑ P(i,j)，对所有 **i < j**

三者之和为 1。

### 4.4 总进球数概率

总进球 **n** = i + j：

$$
P(\text{总进球}=n) = \sum_{i+j=n} P(\text{主}=i) \times P(\text{客}=j)
$$

对 n=0,1,2,… 分别求和。例如 n=2 时，(i,j)=(2,0),(1,1),(0,2) 三种。

### 4.5 大 / 小 2.5 球

- **大 2.5 球** = P(总进球≥3) = P(3)+P(4)+P(5)+…
- **小 2.5 球** = P(总进球≤2) = P(0)+P(1)+P(2)

## 五、比分选取中的二次加权（简述）

在不修改泊松公式的前提下，可以在**选比分**阶段，用「基本面 + 盘口」对比分做一层二次加权：

1. 第一层：泊松给出**原始比分概率表** P(i:j)，只由 λ₁、λ₂ 决定，不掺盘感。
2. 第二层：根据球队风格（防守 / 对攻）、让球盘（强势上盘盘）、大小球盘口（偏大 / 偏小）等，为不同比分设置简易权重 w(i:j)：
   - 贴合基本面 / 盘口的比分 → w 稍大（如 1.2）；
   - 与基本面 / 盘口明显冲突的比分 → w 稍小（如 0.8）；
   - 中性比分 → w = 1。
3. 计算综合得分 S(i:j) = P(i:j) × w(i:j)，按 S 排序，选出 2～4 个主攻比分；如有需要可再归一化 S 为概率。

详细的比分建模与权重使用说明见 `docs/比分建模与投注手册.md`。

## 六、完整计算清单（按顺序执行）

1. **整理数据**：主队主场的得/失、近6 得/失；客队客场的得/失、近6 得/失 → 算出各场均。
2. **基础 λ**：用「3.2」公式，仅主队主场 + 客队客场，算 λ₁、λ₂。
3. **合成攻防**：用「3.3」的 0.7/0.3 加权，得到主客各的合成进攻、合成防守。
4. **合成 λ**：用合成攻防再代入「3.2」中的 λ 公式，得到合成 λ₁、λ₂。
5. **伤停修正**：按「3.4」对 λ₁、λ₂ 加减，得到**最终 λ₁、λ₂**。
6. **预期总进球**：λ₁ + λ₂。
7. **泊松分布**：用最终 λ₁、λ₂ 算 P(主=k)、P(客=k)，k=0～5 或 6。
8. **比分**：P(i,j)=P(主=i)×P(客=j)，列成矩阵。
9. **胜平负**：按 i>j / i=j / i<j 对 P(i,j) 分类求和。
10. **总进球**：按 n=i+j 对 P(i,j) 分组求和，得 P(总进球=n)。
11. **大/小 2.5**：P(≥3) 与 P(≤2)。

## 七、样本与主客场说明

- **主队主场、客队客场**：建议至少 **5～6 场** 再单独用于算 λ；不足时可混入「总」或降低主客场权重。
- **近 6 场**：固定取最近 6 场（不足 6 场则有几场算几场，可适当降低近6 权重）。
- 基础 λ 若**已用主队主场 + 客队客场**，则主客场因素已包含，一般**不再**额外乘「主场优势系数」，避免重复。

## 八、赔率与泊松概率的比较

### 7.1 计算公式

- **庄家隐含概率（未归一）**：十进制赔率对应的概率 = **1 / 赔率**
  - 例：主胜赔率 1.57 → 主胜隐含概率 = 1/1.57 ≈ 63.7%。
- **抽水（overround）**：主胜、平、客胜三个 1/赔率 之和通常 > 1，多出的部分即庄家抽水。
  - 例：1/主 + 1/平 + 1/客 = 108% → 抽水约 8%。
- **归一化（可选）**：若要去掉抽水再比，可把三个 1/赔率 分别除以它们的和，使三者之和 = 100%。

### 7.2 怎么比较

用**泊松算出的胜/平/负概率**与**庄家隐含概率**逐项对比：

| 比较结果                    | 含义                                                                                         |
| --------------------------- | -------------------------------------------------------------------------------------------- |
| 泊松概率 **>** 庄家隐含概率 | 该结果在你模型里被庄家**低估**，可能存在**价值**。                                           |
| 泊松概率 **<** 庄家隐含概率 | 该结果在你模型里被庄家**高估**，无价值。                                                     |
| 泊松概率 **≈** 庄家隐含概率 | 与市场看法接近，说明赔率已经按你的看法「定价」，一般没有额外优势，只是说明你没看错方向而已。 |

更直观的理解方式是：\n\n- 先用泊松算出真实概率 p；\n- 你的「公平赔率」就是 **1/p**；\n- 庄家给的实际赔率记作 O：\n - 若 **O > 1/p**，说明这边赔率比你心里的公平价更高，\n - 等价于 **p × O > 1**，期望值为正，这一边对你来说就叫「便宜」，才值得考虑下注；\n - 若 **O ≤ 1/p**，等价于 **p × O ≤ 1**，长期下注是负期望，这一边就是「贵」，不下注。\n\n**期望值粗算**：若用你的泊松概率 p 下注该结果，期望回报 ≈ **p × 赔率 O**。大于 1 表示长期平均每 1 元下注能拿回超过 1 元（正期望），小于 1 表示长期会亏（负期望）。

### 7.3 意义

- **价值判断**：找出「泊松概率 > 隐含概率」的方向，作为潜在价值选项（不保证命中，长期按期望值下注才占优）。
- **校准参考**：若泊松与庄家长期偏差很大，可反推检查 λ 或数据是否合理。
- **统一口径**：比较时要么都用「未归一」隐含概率，要么都用「归一」隐含概率，避免混用。

## 十、低比分修正（Dixon–Coles 经验版草案）

> 目标：在不改 λ 和泊松主结构的前提下，给 0:0、1:0、0:1、1:1 这类比分做轻量级修正，让平局 / 小比分更贴真实数据。

### 10.1 离线拟合思路（以后可精细化）

1. 选定联赛（如西甲），拉过去 2–3 个赛季全部比分。\n2. 用当时的 λ 算法，对每场算出 λ₁、λ₂，并算出普通泊松乘积下的 P_Poisson(i,j)。\n3. 统计 0:0、1:0、0:1、1:1 的实际频率 P_real(i,j)，对比 P_Poisson(i,j)：\n   - f(i,j) ≈ P_real(i,j) / P_Poisson(i,j)。\n4. 得到一组固定的 f(i,j)，记为本联赛的 DC 系数。

当前阶段可以先用经验系数启动，后续再用真实数据更新。

### 10.2 日常使用规则（经验参数版）

1. 先用现有流程算出原始比分概率矩阵 P_raw(i,j)；\n2. 对低比分集合 S={0:0, 1:0, 0:1, 1:1} 乘经验系数：\n   - f(0:0) = 1.10\n   - f(1:0) = 1.05\n   - f(0:1) = 1.05\n   - f(1:1) = 1.10\n   - 其它比分 f(i,j) = 1\n3. 计算未归一化概率：P_tmp(i,j) = P_raw(i,j) × f(i,j)。\n4. 归一化：Z = Σ P_tmp(i,j)，P_corr(i,j) = P_tmp(i,j) / Z。\n5. 后续胜平负 / 总进球 / 大小球 / 比分权重，全部基于 P_corr(i,j) 而不是 P_raw(i,j)。

> 注意：\n> - 这只是经验版起点，真实 f(i,j) 应该用历史数据按联赛拟合；\n> - 一旦拟合出更好的 f(i,j)，只需替换上面这四个系数，其余流程不变。

## 九、公式速查

| 项目         | 公式                                  |
| ------------ | ------------------------------------- |
| 单队进球概率 | P(X=k) = λ^k·e^{-λ}/k!                |
| 基础 λ₁      | (主队主场场均进 + 客队客场场均失) / 2 |
| 基础 λ₂      | (客队客场场均进 + 主队主场场均失) / 2 |
| 合成攻防     | 0.7×主/客场均 + 0.3×近6场均           |
| 比分         | P(i:j) = P(主=i)×P(客=j)              |
| 预期总进球   | λ₁ + λ₂                               |
| 主胜/平/客胜 | ∑P(i,j) 其中 i>j / i=j / i<j          |
| 总进球=n     | ∑\_{i+j=n} P(主=i)×P(客=j)            |
| 大 2.5       | P(总进球≥3)                           |
| 隐含概率     | 1 / 赔率（十进制）                    |
| 期望值粗算   | 泊松概率 × 赔率（>1 为正期望）        |

以上即为从数据到胜平负、预期总进球、总进球数、比分，以及赔率比较的完整公式与步骤总结。
