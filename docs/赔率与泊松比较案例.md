# 赔率与泊松比较：何时值得下注（案例详解）

**判断标准**：若**泊松概率 > 庄家隐含概率**，且**期望值 = 泊松概率 × 赔率 > 1**，则该方向在你模型里**可能有价值**，值得重点考虑；反之则不值得下注。

下面用三个案例说明：怎么算、怎么比、怎么下结论。

---

## 案例一：布伦特福德 vs 狼队（真实赔率）

**泊松结果**（来自 `data/004-布伦特福德-vs-狼队-泊松分析.md`）

- 主胜 **54.2%**，平 **24.2%**，客胜 **21.6%**

**庄家赔率**：主胜 1.57 / 平 3.80 / 客胜 5.50

### 步骤 1：赔率 → 隐含概率

| 结果 | 赔率 | 隐含概率 = 1/赔率  |
| ---- | ---- | ------------------ |
| 主胜 | 1.57 | 1/1.57 = **63.7%** |
| 平   | 3.80 | 1/3.80 = **26.3%** |
| 客胜 | 5.50 | 1/5.50 = **18.2%** |

### 步骤 2：逐项比较

| 结果 | 泊松概率 | 庄家隐含 | 谁高？     | 期望值 = 泊松×赔率        |
| ---- | -------- | -------- | ---------- | ------------------------- |
| 主胜 | 54.2%    | 63.7%    | 庄家高     | 54.2%×1.57 = **0.85** < 1 |
| 平   | 24.2%    | 26.3%    | 庄家高     | 24.2%×3.80 = **0.92** < 1 |
| 客胜 | 21.6%    | 18.2%    | **泊松高** | 21.6%×5.50 = **1.19** > 1 |

### 结论（是否值得下注）

- **主胜、平**：泊松 < 庄家隐含，期望值 < 1 → **不值得下注**。
- **客胜**：泊松 21.6% > 庄家隐含 18.2%，且期望值 1.19 > 1 → **在你模型里客胜有潜在价值，是唯一值得考虑的方向**（实际下注还需结合仓位与风险偏好）。

**含义**：庄家把主胜开得很热（隐含 63.7%），而你只有 54.2%，主胜明显偏贵；客胜庄家只给 18.2% 概率，你给 21.6%，客胜赔率偏甜，所以“值得下注”的是**客胜**。

## 案例二：赫尔蒙德 vs 坎布尔（假设赔率：主队被低估）

**泊松结果**（来自 `data/001-赫尔蒙德-vs-坎布尔-泊松分析.md`）

- 主胜 **32.5%**，平 **24.8%**，客胜 **42.7%**

**假设庄家赔率**（客队排名高、更被看好）：主胜 3.00 / 平 3.40 / 客胜 2.20

### 步骤 1：赔率 → 隐含概率

| 结果 | 赔率 | 隐含概率 = 1/赔率  |
| ---- | ---- | ------------------ |
| 主胜 | 3.00 | 1/3.00 = **33.3%** |
| 平   | 3.40 | 1/3.40 = **29.4%** |
| 客胜 | 2.20 | 1/2.20 = **45.5%** |

### 步骤 2：逐项比较

| 结果 | 泊松概率 | 庄家隐含 | 谁高？ | 期望值 = 泊松×赔率        |
| ---- | -------- | -------- | ------ | ------------------------- |
| 主胜 | 32.5%    | 33.3%    | 接近   | 32.5%×3.00 = **0.98** < 1 |
| 平   | 24.8%    | 29.4%    | 庄家高 | 24.8%×3.40 = **0.84** < 1 |
| 客胜 | 42.7%    | 45.5%    | 庄家高 | 42.7%×2.20 = **0.94** < 1 |

### 结论

- 三个方向都是泊松 ≤ 庄家隐含，期望值都 < 1 → **没有明显值得下注的方向**；若一定要选，主胜最接近“不亏”（0.98），但仍略负期望。

**换一组假设**（主队被明显低估）：主胜 2.80 / 平 3.30 / 客胜 2.50

- 主胜隐含 = 35.7%，泊松 32.5% < 35.7% → 主胜仍无价值。
- 若改成主胜 **3.20** / 平 3.30 / 客胜 2.15：主胜隐含 = 31.25%，泊松 **32.5% > 31.25%**，期望值 32.5%×3.20 = **1.04** > 1 → **主胜在你模型里值得下注**，即“主队被低估、主胜有价值”的案例。

## 案例三：巴列卡诺 vs 莱万特（假设赔率：主略被低估）

**泊松结果**（来自 `data/006-巴列卡诺-vs-莱万特-泊松分析.md`）

- 主胜 **50.8%**，平 **26.2%**，客胜 **23.0%**

**假设庄家赔率**：主胜 2.05 / 平 3.25 / 客胜 3.50

### 步骤 1：赔率 → 隐含概率

| 结果 | 赔率 | 隐含概率 = 1/赔率  |
| ---- | ---- | ------------------ |
| 主胜 | 2.05 | 1/2.05 = **48.8%** |
| 平   | 3.25 | 1/3.25 = **30.8%** |
| 客胜 | 3.50 | 1/3.50 = **28.6%** |

### 步骤 2：逐项比较

| 结果 | 泊松概率 | 庄家隐含 | 谁高？     | 期望值 = 泊松×赔率        |
| ---- | -------- | -------- | ---------- | ------------------------- |
| 主胜 | 50.8%    | 48.8%    | **泊松高** | 50.8%×2.05 = **1.04** > 1 |
| 平   | 26.2%    | 30.8%    | 庄家高     | 26.2%×3.25 = **0.85** < 1 |
| 客胜 | 23.0%    | 28.6%    | 庄家高     | 23.0%×3.50 = **0.81** < 1 |

### 结论

- **主胜**：泊松 50.8% > 庄家隐含 48.8%，期望值 1.04 > 1 → **主胜在你模型里有潜在价值，值得考虑下注**。
- 平、客胜：期望值 < 1 → 不值得。

**含义**：庄家把主胜概率压到 48.8%，你给 50.8%，主胜赔率 2.05 对你来说略“甜”，所以主胜是值得比较的方向。

---

## 总结：怎么比较才叫“值得下注”

1. **算庄家隐含概率**：每个结果都用 1/赔率（十进制）。
2. **和泊松概率比**：
   - **泊松概率 > 隐含概率** → 该结果在你模型里被庄家**低估**，有可能有价值。
   - **泊松概率 ≤ 隐含概率** → 该结果偏贵，不值得下注。
3. **算期望值**：**泊松概率 × 赔率**。
   - **> 1** → 正期望，在你模型里**值得下注**（仍要结合仓位与风险）。
   - **≤ 1** → 负期望，不值得。
4. **只对“泊松 > 隐含 且 期望值 > 1”的方向考虑下注**；若三个结果都不满足，则本场不做价值下注。

| 案例                  | 泊松来源 | 有价值方向                        | 原因简述                               |
| --------------------- | -------- | --------------------------------- | -------------------------------------- |
| 一 布伦特福德 vs 狼队 | 真实赔率 | **客胜**                          | 泊松 21.6% > 隐含 18.2%，期望 1.19 > 1 |
| 二 赫尔蒙德 vs 坎布尔 | 假设赔率 | 主胜（仅当主胜赔率开到约 3.2 时） | 主队被低估时，主胜才有正期望           |
| 三 巴列卡诺 vs 莱万特 | 假设赔率 | **主胜**                          | 泊松 50.8% > 隐含 48.8%，期望 1.04 > 1 |

_案例二、三的赔率为假设，仅用于演示比较方法；实际以当前市场赔率为准。_
