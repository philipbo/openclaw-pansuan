# 球队风格判定手册（简版）

> 目标：给自己一套简单、统一的规则，把每支球队粗分成「偏大球对攻 / 中性 / 偏小球防守」，方便在泊松比分和大小球上做权重，而不需要看很多录像。

---

## 一、你到底需要什么级别的「风格」？

在你现在的体系里，球队风格主要用在两处：

1. **大小球方向感**：这队更容易踢出大球还是小球；
2. **比分权重**：
   - 防守队 → 0:0、1:0、1:1 这类比分权重大一点；
   - 对攻队 → 2:1、2:2、3:2 这类比分权重大一点。

所以你不需要极其细致的战术标签，只需要粗分成三类：

- **类型 A：偏大球 / 对攻队**
- **类型 B：中性队**
- **类型 C：偏小球 / 防守队**

剩下的都交给泊松和盘口来处理就够了。

---

## 二、从硬数据判定球队风格

以联赛数据为主，优先看：

1. **场均进球（总 / 主 / 客 / 近 6 场）**
2. **场均失球（总 / 主 / 客 / 近 6 场）**
3. **总进球 = 进球 + 失球（总 / 近 6 场）**
4. **大小球盘长期开在哪里**（2 / 2.25 / 2.5 / 2.75 / 3…）

### 2.1 判定总进球倾向

设联赛平均总进球为 G
a) league（例如西甲 ~2.5，荷乙 ~3.0）：

- 若某队 **总进球（得+失）明显高于 G
a) league +0.3~0.5**：
  - 倾向 **类型 A：偏大球 / 对攻队**
- 若某队 **总进球（得+失）明显低于 G
a) league −0.3~0.5**：
  - 倾向 **类型 C：偏小球 / 防守队**
- 介于两者之间：
  - **类型 B：中性队**

可以简单记为：

- 总进球 **≥2.8～3.0**：偏大球（按联赛调节门槛）；
- 总进球 **≤2.2～2.3**：偏小球；
- 其余：中性。

### 2.2 进攻 / 防守强弱

用场均进球 / 失球来做一个 3×3 的判定：

- **进攻强**：场均进球 > 联赛平均 +0.2~0.3；
- **进攻弱**：场均进球 < 联赛平均 −0.2~0.3；
- 其余：中。

- **防守强**：场均失球 < 联赛平均 −0.2~0.3；
- **防守弱**：场均失球 > 联赛平均 +0.2~0.3；
- 其余：中。

组合举例：

- 攻强 + 防弱 → 打开对攻的典型人选；
- 攻弱 + 防强 → 典型防守反击 / 小球队；
- 攻强 + 防强 → 强队压制型，多见 2:0、2:1；
- 攻弱 + 防弱 → 输少当赢型，中立略偏大球（丢球导致拉扯）。

### 2.3 主客场 & 近况修正

- 主场得失球明显高于客场 → 主场敢攻、客场保守；
- 近 6 场得失球与赛季平均差异大：
  - 近 6 场总进球明显上升 → 最近更开放 / 防守崩；
  - 近 6 场总进球明显下降 → 最近更保守 / 防守回稳。

可以简单给「近期修正标记」：

- 近 6 场总进球 > 赛季总进球 +0.3：**短期偏大球**；
- 近 6 场总进球 < 赛季总进球 −0.3：**短期偏小球**。

## 三、借助盘口反推风格

当数据不够细，也可以用盘口长期形态来反推：

### 3.1 长期大小球盘

- 经常开 **3 球、3.25 甚至更高**：
  - 说明市场普遍认为这支队的大球率高，对攻 / 防守差；
- 经常开 **2 球、2/2.5**：
  - 说明多被视为小球队，节奏慢、防守稳或进攻差。

### 3.2 让球盘表现

- 一支队经常在半一、一球盘下还能稳稳赢盘：
  - 往往是攻强、防相对不错，有「赢球赢盘」能力的强队；
- 一支队经常只以一球小胜、或者让半一经常输盘：
  - 进攻不追杀，多见控场但效率一般的球队。

这些信息配合硬数据使用即可，不必单独当判定依据。

---

## 四、把风格归类为 A / B / C

最后给每队一个简单标签：

- **类型 A：偏大球 / 对攻队**
  - 特征：总进球高（联赛平均+）、进攻强、防守弱或一般，大小盘多在 2.75+/3；
- **类型 B：中性队**
  - 特征：总进球接近联赛均值，攻守都中等或一攻一守；
- **类型 C：偏小球 / 防守队**
  - 特征：总进球低（联赛平均−）、防守强、进攻一般或偏弱，大小盘多在 2/2.25、2.25/2.5。

> 可以把这个标签记在你自己的球队笔记里，后面选比分和大小球时直接查。

## 五、风格对比分和大小球的具体影响

### 5.1 大小球方向

- 类型 A 队：
  - 对阵同样偏大的队 → 更偏大球（配合泊松 λ 调整）；
  - 对阵偏小队 → 方向不一定绝对大，要看谁主导节奏。
- 类型 C 队：
  - 双方都偏小 → 小球权重更高；
  - 对阵偏大队 → 中性，更多看对手掌控力和盘口。

### 5.2 比分权重（用在 w(i:j) 上）

- 类型 A 队：
  - 若你主看其获胜：2:1、3:1、3:2、2:2 这类比分 w 稍大；
  - 0:0、1:0、1:1 一类 w 稍小，但仍保留做防守比分。  
- 类型 C 队：
  - 0:0、1:0、1:1、2:0 这类小比分 w 稍大；
  - 3:2、4:3 一类比分 w 稍小或干脆忽略。

> 细则和权重具体怎么打，见 `docs/比分建模与投注手册.md` 中关于 w(i:j) 的部分。

## 六、最低可执行版本（给自己一个懒人流程）

每次分析一支队，只需做三步打标签：

1. 看赛季总进球（得+失）+ 近 6 场总进球：
   - 明显高于联赛平均 → 先标记为「偏大球」；
   - 明显低于联赛平均 → 先标记为「偏小球」。
2. 用场均进球 / 失球粗分攻强 / 防强：
   - 攻强+防弱 → 强对攻；
   - 攻弱+防强 → 小球防守队；
3. 用长期大小盘位置 / 常见比分印象做微调：
   - 经常 3 球以上盘口 → 再确认偏大球；
   - 经常 2/2.25 → 再确认偏小球。

最后写在你自己的笔记里：

- 「队名：类型 A / B / C，攻强/中/弱，防强/中/弱」

后面泊松选比分、选大小球，只要查这个标签，就知道该往哪一边偏权重。EOF
