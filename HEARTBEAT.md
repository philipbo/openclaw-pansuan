# Heartbeat 检查清单

读取当日 memory 文件（`memory/{今天日期}.md`），按以下清单检查。全部通过则回复 `HEARTBEAT_OK`，有需要处理的事项则执行并推送。

## 赛前分析补漏

条件：**17:00–销售截止** 且当日 memory 中：

- 有 `## 跳过分析` → 跳过
- 有 `## 推荐` → 跳过
- 有 `## 深度分析` 但无 `## 推荐` → 从推荐步骤续做
- 有 `## 初筛结果` 但无 `## 深度分析` → 从深度分析步骤续做
- 什么都没有 → 执行赛前全流程

## 赛后复盘补漏

条件：**当前时间 ≥ 09:30** 且 `memory/{昨天日期}.md` 中无 `## 赛后复盘`：

- 执行赛后复盘流程
- 同时检查前 3 天 memory 中的「待复盘」场次
