# Heartbeat 检查清单

读取当日 memory 文件（`memory/{今天日期}.md`），按以下清单检查。全部通过则回复 `HEARTBEAT_OK`，有需要处理的事项则执行并推送。

## 赛前分析补漏

条件：当前时间 **≥ 17:00** 且 **未过当日竞彩销售截止**（周一至五 22:00、周六日 23:00）且当日 memory 中：

- 有 `## 跳过分析` → 跳过
- 有 `## 推荐` → 跳过
- 有 `## 赛程抓取` 且场次为 0 且无 `## 初筛结果` → 跳过（今日无比赛）
- 有 `## 初筛结果` 且通过场次为 0 且无 `## 深度分析` → 跳过（视为已完成）
- 有 `## 深度分析` 但无 `## 推荐` → 从推荐步骤续做
- 有 `## 初筛结果` 但无 `## 深度分析` → 从深度分析步骤续做
- 什么都没有 → 执行赛前全流程

## 赛后复盘补漏

条件：**当前时间 ≥ 09:30** 且 `memory/{昨天日期}.md` 中无 `## 赛后复盘`：

- 执行赛后复盘流程（复盘昨天）
- 同时检查前 3 天 memory 中的「待复盘」场次

额外检查：如果 `memory/{前天日期}.md` 存在且无 `## 赛后复盘` → 一并补做前天的复盘
