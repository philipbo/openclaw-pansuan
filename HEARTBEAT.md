# 定时检查清单

每次心跳（30 分钟）执行以下检查，**通过读取当日 memory 文件判断各阶段是否已完成**，避免重复执行：

## 赛前分析

1. 读取 `memory/{今天日期}.md`，检查已完成的阶段：
   - 有 `## 早波推荐` → 早波已完成
   - 有 `## 晚波推荐（全天汇总）` → 晚波已完成
   - 有 `## 推荐（HH:MM）` → 手动触发已完成
2. 判断当前应执行什么：
   - **11:00–16:59 且无早波推荐** → 执行早波赛前全流程
   - **17:00–销售截止 且无晚波推荐** → 执行晚波赛前全流程
   - **其他情况** → 跳过，不重复执行
3. 如果当日 memory 文件不存在（首次心跳）→ 根据当前时间段执行对应波次

## 赛后复盘

- 读取 `memory/{昨天日期}.md`，检查是否有 `## 赛后复盘`
- **无复盘记录且当前时间 ≥ 09:30** → 执行赛后复盘流程
- 同时检查前 3 天 memory 中标记为「待复盘」的场次，一并处理
