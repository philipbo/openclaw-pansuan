# 定时检查清单

每次心跳（30 分钟）执行以下检查，**通过读取当日 memory 文件判断各阶段是否已完成**，避免重复执行：

## 赛程同步

1. 读取 `memory/{今天日期}.md`，检查是否有 `## 赛程抓取`
2. **11:00–11:30 且无赛程抓取记录** → 执行赛程抓取，推送赛程列表给龙王
3. 龙王手动说「同步赛程」「同步比赛」「今天有什么比赛」→ 无论是否已有记录，都重新抓取并推送（覆盖写入）

## 赛前分析

1. 读取 `memory/{今天日期}.md`，按优先级检查以下标记：
   - 有 `## 跳过分析` → **本日不执行赛前分析**，直接跳过（龙王主动说「今天不分析」「今天休息」时写入此标记）
   - 有 `## 推荐` → 已完成，跳过
   - 有 `## 深度分析` 但无 `## 推荐` → **从推荐步骤续做**：读取已分析的场次，对未分析的场次继续深度分析，全部完成后执行 recommendation
   - 有 `## 初筛结果` 但无 `## 深度分析` → **从深度分析步骤续做**：读取初筛通过的场次列表，启动 deep-analysis
   - 什么都没有 → 执行赛前全流程（抓取→初筛→深度分析→精选→推荐）

2. **触发条件**：17:00–销售截止 且未命中上述任何"跳过"条件

3. 如果当日 memory 文件不存在（首次心跳）→ 根据当前时间段执行对应流程

### 跳过机制

当龙王说「今天不分析」「今天不推荐了」「今天休息」时：

- 在 `memory/{今天日期}.md` 中写入 `## 跳过分析`
- 心跳检测到此标记后不再触发赛前分析
- 龙王可以随时说「还是分析吧」「恢复分析」来删除此标记，恢复心跳触发

## 赛后复盘

- 读取 `memory/{昨天日期}.md`，检查是否有 `## 赛后复盘`
- **无复盘记录且当前时间 ≥ 09:30** → 执行赛后复盘流程
- 同时检查前 3 天 memory 中标记为「待复盘」的场次，一并处理
