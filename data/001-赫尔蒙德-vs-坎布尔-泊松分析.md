## 二、基础 λ（主队主场 + 客队客场）

公式：
- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：
- λ₁ = (1.6875 + 1.2857) / 2 = **1.4866**
- λ₂ = (1.8571 + 1.5625) / 2 = **1.7098**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队
- 合成进攻 = 0.7×1.6875 + 0.3×1.0000 = 1.1813 + 0.3000 = **1.4813**
- 合成防守 = 0.7×1.5625 + 0.3×1.6667 = 1.0938 + 0.5000 = **1.5938**

### 客队
- 合成进攻 = 0.7×1.8571 + 0.3×1.6667 = 1.3000 + 0.5000 = **1.8000**
- 合成防守 = 0.7×1.2857 + 0.3×1.0000 = 0.9000 + 0.3000 = **1.2000**

### 用合成攻防再算 λ
- λ₁ = (1.4813 + 1.2000) / 2 = **1.3406**
- λ₂ = (1.8000 + 1.5938) / 2 = **1.6969**

**最终 λ**（无伤停修正）：
- **λ₁（赫尔蒙德）= 1.34**
- **λ₂（坎布尔）= 1.70**
- **预期总进球 = 1.34 + 1.70 = 3.04** 球
