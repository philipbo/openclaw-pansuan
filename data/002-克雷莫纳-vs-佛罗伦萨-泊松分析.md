## 二、基础 λ（主队主场 + 客队客场）

公式：

- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：

- λ₁ = (0.8462 + 1.6429) / 2 = **1.2446**
- λ₂ = (0.8571 + 1.3077) / 2 = **1.0824**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队

- 合成进攻 = 0.7×0.8462 + 0.3×0.3333 = 0.5923 + 0.1000 = **0.6923**
- 合成防守 = 0.7×1.3077 + 0.3×1.8333 = 0.9154 + 0.5500 = **1.4654**

### 客队

- 合成进攻 = 0.7×0.8571 + 0.3×1.0000 = 0.6000 + 0.3000 = **0.9000**
- 合成防守 = 0.7×1.6429 + 0.3×1.3333 = 1.1500 + 0.4000 = **1.5500**

### 用合成攻防再算 λ

- λ₁ = (0.6923 + 1.5500) / 2 = **1.1212**
- λ₂ = (0.9000 + 1.4654) / 2 = **1.1827**

**最终 λ**（无伤停修正）：

- **λ₁（克雷莫纳）= 1.12**
- **λ₂（佛罗伦萨）= 1.18**
- **预期总进球 = 1.12 + 1.18 = 2.30** 球
