## 二、基础 λ（主队主场 + 客队客场）

公式：

- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：

- λ₁ = (1.1538 + 1.6154) / 2 = **1.3846**
- λ₂ = (1.1538 + 0.7692) / 2 = **0.9615**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队

- 合成进攻 = 0.7×1.1538 + 0.3×1.6667 = 0.8077 + 0.5000 = **1.3077**
- 合成防守 = 0.7×0.7692 + 0.3×0.8333 = 0.5385 + 0.2500 = **0.7885**

### 客队

- 合成进攻 = 0.7×1.1538 + 0.3×0.8333 = 0.8077 + 0.2500 = **1.0577**
- 合成防守 = 0.7×1.6154 + 0.3×1.8333 = 1.1308 + 0.5500 = **1.6808**

### 用合成攻防再算 λ

- λ₁ = (1.3077 + 1.6808) / 2 = **1.4943**
- λ₂ = (1.0577 + 0.7885) / 2 = **0.9231**

**最终 λ**（无伤停修正）：

- **λ₁（巴列卡诺）= 1.49**
- **λ₂（莱万特）= 0.92**
- **预期总进球 = 1.49 + 0.92 = 2.41** 球
