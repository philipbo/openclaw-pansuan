## 二、基础 λ（主队主场 + 客队客场）

公式：

- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：

- λ₁ = (1.7143 + 1.5000) / 2 = **1.6072**
- λ₂ = (0.3571 + 1.0714) / 2 = **0.7143**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队

- 合成进攻 = 0.7×1.7143 + 0.3×1.5000 = 1.2000 + 0.4500 = **1.6500**
- 合成防守 = 0.7×1.0714 + 0.3×1.3333 = 0.7500 + 0.4000 = **1.1500**

### 客队

- 合成进攻 = 0.7×0.3571 + 0.3×1.1667 = 0.2500 + 0.3500 = **0.6000**
- 合成防守 = 0.7×1.5000 + 0.3×1.1667 = 1.0500 + 0.3500 = **1.4000**

### 用合成攻防再算 λ

- λ₁ = (1.6500 + 1.4000) / 2 = **1.5250**
- λ₂ = (0.6000 + 1.1500) / 2 = **0.8750**

**最终 λ**（无伤停修正）：

- **λ₁（布伦特福德）= 1.53**
- **λ₂（狼队）= 0.88**
- **预期总进球 = 1.53 + 0.88 = 2.41** 球

## 十、赔率与泊松概率对比

**庄家赔率（示例）**：主胜 1.57 / 平 3.80 / 客胜 5.50

### 赔率 → 隐含概率

**原始隐含概率** = 1 / 赔率：

| 结果     | 赔率 | 1/赔率 | 隐含概率（未归一） |
| -------- | ---- | ------ | ------------------ |
| 主胜     | 1.57 | 1/1.57 | **63.69%**         |
| 平       | 3.80 | 1/3.80 | **26.32%**         |
| 客胜     | 5.50 | 1/5.50 | **18.18%**         |
| **合计** | —    | —      | **108.19%**        |

合计大于 100% 的部分为庄家抽水（约 8.19%）。

### 去抽水后的隐含概率（归一）

| 结果     | 未归一  | 归一后                    |
| -------- | ------- | ------------------------- |
| 主胜     | 63.69%  | 63.69÷108.19 = **58.87%** |
| 平       | 26.32%  | 26.32÷108.19 = **24.33%** |
| 客胜     | 18.18%  | 18.18÷108.19 = **16.80%** |
| **合计** | 108.19% | **100%**                  |

### 与泊松概率对比

| 结果 | 泊松概率  | 庄家（未归一） | 庄家（归一） | 对比                                     |
| ---- | --------- | -------------- | ------------ | ---------------------------------------- |
| 主胜 | **54.2%** | 63.7%          | 58.9%        | 泊松 < 庄家 → 主胜无价值                 |
| 平   | **24.2%** | 26.3%          | 24.3%        | 接近                                     |
| 客胜 | **21.6%** | 18.2%          | 16.8%        | 泊松 > 庄家 → 客胜可能被低估，有潜在价值 |

### 期望值粗算（按泊松概率下注）

- 主胜：54.2% × 1.57 ≈ **0.85** < 1
- 平：24.2% × 3.80 ≈ **0.92** < 1
- 客胜：21.6% × 5.50 ≈ **1.19** > 1 → 仅从期望值看，客胜方向有正期望

**说明**：赔率为示例，实际分析请以当前市场赔率为准。
