## 二、基础 λ（主队主场 + 客队客场）

公式：

- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：

- λ₁ = (1.0556 + 1.2778) / 2 = **1.1667**
- λ₂ = (1.5556 + 1.0556) / 2 = **1.3056**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队

- 合成进攻 = 0.7×1.0556 + 0.3×1.5000 = 0.7389 + 0.4500 = **1.1889**
- 合成防守 = 0.7×1.0556 + 0.3×1.3333 = 0.7389 + 0.4000 = **1.1389**

### 客队

- 合成进攻 = 0.7×1.5556 + 0.3×1.5000 = 1.0889 + 0.4500 = **1.5389**
- 合成防守 = 0.7×1.2778 + 0.3×1.5000 = 0.8945 + 0.4500 = **1.3445**

### 用合成攻防再算 λ

- λ₁ = (1.1889 + 1.3445) / 2 = **1.2667**
- λ₂ = (1.5389 + 1.1389) / 2 = **1.3389**

**最终 λ**（无伤停修正）：

- **λ₁（朴茨茅斯）= 1.27**
- **λ₂（德比郡）= 1.34**
- **预期总进球 = 1.27 + 1.34 = 2.61** 球
