## 二、基础 λ（主队主场 + 客队客场）

公式：

- λ₁ = (主队主场场均进 + 客队客场场均失) / 2
- λ₂ = (客队客场场均进 + 主队主场场均失) / 2

代入：

- λ₁ = (1.5000 + 0.8333) / 2 = **1.1667**
- λ₂ = (1.0000 + 0.9167) / 2 = **0.9584**

---

## 三、掺入近六场（0.7 主客场 + 0.3 近六场）

### 主队

- 合成进攻 = 0.7×1.5000 + 0.3×1.3333 = 1.0500 + 0.4000 = **1.4500**
- 合成防守 = 0.7×0.9167 + 0.3×1.1667 = 0.6417 + 0.3500 = **0.9917**

### 客队

- 合成进攻 = 0.7×1.0000 + 0.3×1.5000 = 0.7000 + 0.4500 = **1.1500**
- 合成防守 = 0.7×0.8333 + 0.3×1.3333 = 0.5833 + 0.4000 = **0.9833**

### 用合成攻防再算 λ

- λ₁ = (1.4500 + 0.9833) / 2 = **1.2167**
- λ₂ = (1.1500 + 0.9917) / 2 = **1.0708**

**最终 λ**（无伤停修正）：

- **λ₁（阿纳西）= 1.22**
- **λ₂（特鲁瓦）= 1.07**
- **预期总进球 = 1.22 + 1.07 = 2.29** 球
