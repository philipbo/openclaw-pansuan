# 初盘赔率与泊松价值分析（001～006）

**说明**：基于各场泊松分析文档中的胜平负概率，与初盘赔率对比，按统一流程判断「哪个结果值得买」。  
**赔率**：主胜 / 平 / 客胜（十进制初盘）。

---

## 一、统一流程（学习用）

对每一场按下面 4 步做一遍：

### 步骤 1：写出泊松概率

从该场的泊松分析文档「胜平负」汇总表取数：

- 主胜概率 P主、平局概率 P平、客胜概率 P客（百分比，用小数参与计算时除以 100）。

### 步骤 2：赔率 → 庄家隐含概率

对每个结果：**隐含概率 = 1 / 赔率**（十进制）。

可选：把主/平/客三个隐含概率相加，若大于 1，多出的部分即庄家抽水；若要做归一化，再各自除以该总和。

### 步骤 3：逐项比较 + 算期望值

对主胜、平、客胜分别做：

| 检查项 | 做法                                                                         |
| ------ | ---------------------------------------------------------------------------- |
| 谁高   | 泊松概率 vs 庄家隐含概率：泊松 **>** 隐含 → 该结果在你模型里被庄家**低估**。 |
| 期望值 | **期望值 = 泊松概率 × 赔率**。**> 1** 表示按你的概率下注有正期望。           |

### 步骤 4：下结论——哪个值得买

- **值得买**：**泊松概率 > 庄家隐含概率** 且 **期望值 > 1** 的结果。
- **不值得买**：泊松 ≤ 隐含 或 期望值 ≤ 1 的结果。

同一场可以有一个方向值得买、多个方向、或都没有，依数据而定。

## 三、逐场分析（按流程执行）

### 001 赫尔蒙德 vs 坎布尔

**步骤 2** 隐含概率：主 1/3.90=**25.6%**，平 1/3.80=**26.3%**，客 1/1.75=**57.1%**（和 109.1%）

**步骤 3** 比较与期望值：

| 结果 | 泊松  | 隐含  | 泊松>隐含? | 期望值 = 泊松×赔率    | 期望>1? |
| ---- | ----- | ----- | ---------- | --------------------- | ------- |
| 主胜 | 32.5% | 25.6% | ✓          | 32.5%×3.90 = **1.27** | ✓       |
| 平   | 24.8% | 26.3% | ✗          | 24.8%×3.80 = 0.94     | ✗       |
| 客胜 | 42.7% | 57.1% | ✗          | 42.7%×1.75 = 0.75     | ✗       |

**步骤 4 结论**：**主胜** 值得买（泊松高于隐含，期望值 1.27 > 1）。平、客胜不值得。
