# 推荐输出（recommendation）

汇总 deep-analysis 的分析结果，按信心度排序，生成推荐卡片并推送。

## 输入

来自 deep-analysis 的每场比赛综合评估结果。

## 执行步骤

### 步骤 1：筛选可推荐场次

从深度分析结果中筛选：

- **信心度为「高」或「中」的场次** → 纳入推荐
- **信心度为「低」的场次** → 不推荐，但可列为「观察场次」
- 如果所有场次信心度都低 → 如实告知「今日无高信心推荐，建议观望」

### 步骤 2：排序

按以下优先级排序：

1. 信心度（高 > 中）
2. 联赛层级（高 > 中）
3. 维度一致性（同向维度越多越靠前）

### 步骤 3：生成分析报告

每场推荐包含**完整分析过程 + 结论**，格式如下：

```
📌 [{联赛}] {主队} vs {客队}
⏰ {开球时间}

━━━ 分析过程 ━━━

1️⃣ 基本面分析
{基本面分析内容，用具体数据}

2️⃣ 伤情分析
{伤停情况及对球队影响}

3️⃣ 欧洲指数（胜平负）赔率分析
{表格：各公司初盘→即时盘赔率}
{意图解读：赔率与实力是否匹配、升降原因、是否诱买}

4️⃣ 亚盘赔率分析
{表格：各公司盘口+水位 初盘→即时盘}
{意图解读：让大/让少/正常、是阻还是诱、升降盘原因排查}

5️⃣ 进球数赔率分析
{表格：各公司大小球盘口+水位 初盘→即时盘}
{意图解读：盘口偏高/偏低、诱大球还是诱小球、与场均进球对比}

6️⃣ 赔率合理性评估
{欧指、亚盘、大小球三者交叉验证，一致性判断}

7️⃣ 赔率变动解读
{各公司变动时间线，谁先动、方向是否一致}

8️⃣ 关键点 / 疑点 / 矛盾
{本场最核心的判断要素、不确定项、各维度冲突信号}

9️⃣ 风险提示
{可能推翻判断的因素}

━━━ 结论 ━━━

🎯 推荐：{推荐内容}
⚽ 比分预测：{最看好} > {次看好} > {第三看好}
💪 信心度：{高/中}
📊 核心理由：
  1. {最关键的理由}
  2. {第二关键}
  3. {第三关键（如有）}
⚠️ 风险：{主要风险点}
```

推荐内容的表述规范：

- 胜平负：「主胜」「平局」「客胜」
- 让球：「主队让{X}球 主胜」「主队受让{X}球 主胜」
- 大小球：「大{X.5}球」「小{X.5}球」
- 组合推荐（信心高时可给）：「主胜 + 大2.5球」

**组合推荐统计规则**：组合推荐按玩法拆分计数。例如「主胜 + 大2.5球」算 2 条推荐，分别计入胜平负和大小球的命中统计。但在"推荐场次"维度仍算 1 场，只要其中任一玩法命中即视为该场命中。

### 步骤 4：生成汇总头部

在所有分析报告之前加上汇总：

```
⚽ 盘算今日推荐 | {日期} {星期X}
⏳ 今日购彩截止：{22:00/23:00}（根据星期判断）

今日共筛选 {初筛总数} 场，深度分析 {分析数} 场，推荐 {推荐数} 场。

---
{各场完整分析报告}
---

📋 观察场次（信心度低，不作为推荐）：
- [{联赛}] {主队} vs {客队} — {简要原因}

💡 免责声明：以上为数据分析结论，不构成投注建议。投注需谨慎，量力而行。
```

### 步骤 5：推送

通过 messaging 工具将推荐内容发送给向钱冲。

如果 messaging 不可用：

- 将内容写入当日 memory
- 回复告知向钱冲推荐已生成，可在对话中查看

### 步骤 6：写入记忆

将推荐内容追加写入 `memory/{今天日期}.md`：

```
## 今日推荐

推荐时间：{时间}
推荐场次：{N} 场

{完整推荐内容}

### CLV 追踪数据（复盘用）

| 比赛 | 推荐方向 | 推荐时赔率 | 比赛ID |
| ---- | -------- | ---------- | ------ |
| {主队} vs {客队} | {推荐内容} | {推荐时对应的即时赔率} | {matchId} |
| ... | ... | ... | ... |
```

**重要**：必须记录推荐时的即时赔率（推荐方向对应的赔率），用于赛后复盘时与收盘赔率对比，计算 CLV。

## 特殊场景处理

### 无推荐场景

如果今日无高信心推荐：

```
⚽ 盘算今日推荐 | {日期}

今日共筛选 {N} 场，深度分析 {N} 场。
经分析，今日无高信心推荐场次，建议观望。

📋 观察场次：
- {列出信心度低的场次及原因}

💡 宁可错过，不可乱推。明天见！
```

### 用户追问单场

如果向钱冲问「{某场比赛}怎么看」：

1. 检查是否已分析过（查当日 memory）
2. 如果已分析，直接输出该场分析结果
3. 如果未分析，启动 deep-analysis 单场分析再输出

### 用户要求调整

如果向钱冲说「再多推几场」或「只看英超」：

- 回到初筛结果，按新条件调整
- 必要时对额外场次补做深度分析
- 重新生成推荐

## 输出

推荐内容（文本格式），已推送或已写入记忆。
