# 深度分析（deep-analysis）

对 match-screening 筛选出的每场比赛，按 10 步流程进行深度分析，给出综合倾向、信心度和比分概率排序。

## 输入

支持以下三种输入方式：

### 方式 1：来自初筛流程（默认）

来自 match-screening 的初筛通过场次列表（含比赛 ID 和详情链接）。

### 方式 2：龙王指定比赛 ID

龙王直接提供比赛 ID，例如：「分析 2950979」「帮我看看 2950979 和 2784768」

→ 跳过初筛，直接用比赛 ID 拼接分析页 URL：`https://zq.titan007.com/analysis/{matchId}cn.htm`

### 方式 3：龙王提供分析页 URL

龙王直接粘贴 URL，例如：「分析这场 https://zq.titan007.com/analysis/2950979cn.htm」

→ 从 URL 中提取比赛 ID（正则：`analysis/(\d+)cn`），跳过初筛直接分析。

**方式 2 和 3 的处理规则**：

- 不需要经过初筛，直接进入深度分析
- 可以是任何比赛（不限联赛层级、不限时间）
- 可以同时指定多场比赛（逗号或空格分隔）
- 分析完成后同样进入 recommendation 流程（精选、串关建议等），与初筛流程的后续步骤完全一致
- 如果指定的比赛已经在当日 memory 中有分析记录，提示龙王「此场已分析，是否重新分析？」，避免重复浏览器操作

## 执行流程

对每场比赛依次执行以下步骤：

### 步骤 1：获取分析页数据

**⚠️ 数据更新原则：区分基本面和赔率。**

- **基本面数据**（排名、战绩、伤停、交锋、赛程等）：**同一天内基本不变**。如果当日 memory 中已有该场比赛的基本面数据（之前分析过），直接复用，不必重新抓取分析页。
- **赔率数据**（欧指、亚盘、大小球变化）：**实时变化**，每次分析都必须重新抓取最新赔率（步骤 1.5）。

**判断逻辑**：

1. 检查当日 memory 中是否已有该场比赛的深度分析记录
2. **有记录（重新分析）**→ 跳过步骤 1（基本面已有），直接进入步骤 1.5 刷新赔率数据，然后用新赔率 + 旧基本面重新走分析流程
3. **无记录（首次分析）**→ 执行完整步骤 1 + 步骤 1.5

### 步骤 3：10 步分析过程

每场比赛按以下 10 步依次分析（第 1-9 步为分析，第 10 步为比分建模），**完整展示分析过程**，让龙王能从中学习判断逻辑。

#### 第 1 步：基本面分析

分析要素：

- **球队级别**：识别球队的历史底蕴和档次
  - 超级豪门（皇马、巴萨、拜仁、曼城、利物浦等）：底蕴深厚，大赛经验丰富，逆境翻盘能力强，不可轻易看衰
  - 传统强队（多特、马竞、国米、阿森纳等）：实力强但稳定性不如豪门
  - 中游球队：受状态和阵容影响大，波动性高
  - 保级队/升班马：资源有限，主场拼劲足但客场脆弱
  - 球队级别影响盘口解读：豪门让球少可能是机构看到利空，而非正常定价
- **德比/特殊对决标记**：
  - 同城德比（如米兰德比、曼市德比、国家德比）→ 形势难料，弱队拼劲翻倍，让球参考价值降低
  - 传统宿敌（如凯尔特人 vs 流浪者、加拉塔萨雷 vs 费内巴切）→ 情绪因素放大
  - 争冠直接对话 / 保级生死战 → 战意拉满，比赛强度和不确定性都高
  - 德比/特殊对决需在分析开头标注，因为它会影响后续所有维度的权重
- 双方积分榜排名、目标（争冠/保级/争欧/无欲）
- 近 5-10 场总战绩（胜/平/负），走势（连胜/连败/交替/回升/下滑）
- 主队主场战绩 vs 客队客场战绩
- 场均进球、场均失球，攻防对比
- 赛程密度（一周多赛？间隔天数？轮换风险？）
- 主场因素（主场龙/客场虫？）
- 交锋记录：近 5 次交手结果（主客区分），比分特点（大球多/小球多），是否有克星关系
- **教练风格与球队性格**（Voulgaris 行为模式视角）：
  - 教练战术倾向：进攻型（如克洛普、德泽尔比）vs 防守型（如西蒙尼、孔蒂、穆里尼奥）vs 控球型（如瓜迪奥拉、阿尔特塔）
  - 领先时的行为：保守收缩（换防守球员、五后卫）还是继续进攻？→ 影响大小球和让球判断
  - 落后时的行为：容易崩盘（连丢球）还是愈战愈勇（逆转能力强）？→ 影响让球判断
  - 换人习惯：早换（60分钟前）还是晚换？面对弱队会轮换吗？→ 影响赛程密度评估
  - 球队心态特征：抗压能力（大赛型/小赛型）、主场龙客场虫的心理因素
  - 这些「软信息」是赔率模型难以量化的，如果你掌握得准，就是边际优势
- **均值回归意识**（Benham 核心认知）：
  - 近期连胜 4-5 场的球队 → 大概率被高估，接下来表现回落的概率大于继续连胜
  - 近期连败 4-5 场的球队 → 大概率被低估，反弹随时可能发生
  - 判断方法：近 5 场战绩是否显著偏离赛季整体胜率？如果偏离大，均值回归的力量就强
  - 短期结果 ≠ 真实实力，不追涨杀跌
- **赛程陷阱（Lookahead Spot）**：
  - 检查双方未来五场赛程，关注**下一场是否有大赛**（欧冠淘汰赛、联赛争冠直接对话、杯赛决赛等）
  - 如果下一场是大赛，本场可能出现「心理性轮换」——即使全主力出战，注意力和拼劲也不在本场
  - 这不同于体能轮换（换人），而是精神层面的松懈，更隐蔽也更致命
  - 强队 vs 弱队 + 强队下一场有大赛 = 典型陷阱局，弱队爆冷概率显著提升
- **赛季阶段性特征**：
  - 开赛初期（1-5 轮）：升班马常被低估，数据样本不足，盘口参考价值有限
  - 赛季中段：数据稳定，分析最可靠
  - 圣诞/冬歇前（英超 12 月）：密集赛程，冷门频出，体能和轮换影响大
  - 联赛末段（最后 5 轮）：已保级/已降级/已夺冠的队战意极端分化，「死水比赛」需特殊处理
  - 标记当前赛季阶段，并评估其对本场的影响

输出格式：**开头先标注球队级别和是否德比/特殊对决**，然后用具体数据说话，穿插教练风格、球队性格、均值回归评估和赛程陷阱判断。给出小结判断。

#### 第 2 步：伤情分析

分析要素：

- 主队缺阵球员名单、位置、缺阵原因
- 客队缺阵球员名单、位置、缺阵原因
- **对球队的影响评估**：核心球员（门将、中场核心、主力射手）缺阵影响几何？替补能力如何？
- 是否面临轮换（杯赛/联赛双线作战）

输出格式：列出关键缺阵球员，分析对进攻/防守/整体的影响程度。如无伤停信息，标注「伤停信息缺失，本步骤不计入」。

#### 第 3 步：欧洲指数（胜平负）赔率分析

数据来源：步骤 1.5 抓取的各公司欧指变化。

参考公司：

- 默认：澳彩、皇冠、365、易胜博、**平博**
- 欧洲比赛：加入**威廉希尔**，以威廉为主要参考
- 德国联赛：加入**Interwetten**
- 亚洲联赛：可选加入香港马会

**数据展示**（用表格）：

```
| 公司   | 初盘主胜 | 初盘平局 | 初盘客胜 | 即时主胜 | 即时平局 | 即时客胜 | 变化方向 |
| ------ | -------- | -------- | -------- | -------- | -------- | -------- | -------- |
| 澳彩   | x.xx     | x.xx     | x.xx     | x.xx     | x.xx     | x.xx     | {描述}   |
| ...    | ...      | ...      | ...      | ...      | ...      | ...      | ...      |
```

**分析思路（教学级排查流程）**：

**第一层：初盘合理性判断** — 赔率与实力是否匹配？

- 强队主胜应偏低（<2.00），弱队客胜应偏高（>3.00）
- 如果不匹配，排查原因：伤停？轮换？战意？交锋克制？
- **平局赔率的信号**：平赔维持高位不动（>3.50）= 排除平局；平赔走低（<3.20）= 机构认为可能平局

**第二层：变动排查** — **只要赔率有变化，都必须判断原因**

每一项赔率变动（主胜/平/客胜）逐一排查三种可能：

1. **平衡筹码（被动调整）**：某一结果投注量过大，机构降低该项赔率平衡资金
   - 特征：变动幅度小（0.02-0.05），方向与大众热门一致
   - 例：热门主队大量投注 → 主胜赔率小幅下降 → 正常平衡
2. **机构主动调整（有信息驱动）**：机构基于信息改变判断
   - 特征：变动幅度大（>0.10），或与大众情绪相反
   - 例：主队热门但主胜赔率反升 → 机构不看好主队，有利空信息
3. **诱买操作**：引导散户买某一结果，实际看好其他结果
   - 判断方法：欧指与亚盘交叉验证
   - 例：欧指降主胜（引导买主胜）但亚盘不升盘甚至降盘 → 典型诱买主胜信号

**排查清单（每项赔率变动都要过一遍）**：

```
变动项：{主胜 / 平局 / 客胜}
变动方向：{升(看衰) / 降(看好)}
变动幅度：{小幅(≤0.05) / 中幅(0.05-0.15) / 大幅(>0.15)}
与大众情绪：{同向(可能平衡) / 反向(可能机构主动)}
与亚盘方向是否一致：{一致(信号可靠) / 矛盾(疑似诱买)}
能否用基本面解释：{能 / 不能(暗盘信号)}
→ 结论：{平衡筹码 / 机构主动看好X / 诱买X}
```

**第三层：RLM（逆向线路移动）检测**

RLM 是职业投注者最看重的信号之一：

- **判断大众情绪方向**：热门球队（豪门、连胜队、高排名队）= 大众倾向买的一方
- **观察赔率实际走向**：赔率往哪边移动？
- **两者是否反向**：大众买主胜 → 但主胜赔率反而上升（变贵了）= RLM 信号
- RLM 意味着有**尖锐资金在押另一边**，机构跟着尖锐资金走而不是跟着大众走
- 一旦检测到 RLM，**在第 8 步标记为高权重信号**

```
RLM 检测清单：
大众情绪方向：{买主胜 / 买客胜}（根据球队热度判断）
赔率实际走向：{主胜降(看好) / 主胜升(看衰)}
是否反向：{是(RLM!) / 否(正常)}
→ 如果是 RLM：尖锐资金方向 = {与大众相反的一方}
```

**第四层：平博锚点验证**

- 将其他公司的欧指和**平博(47)**对比
- 平博的收盘赔率 = 尖锐资金共识后的真实概率
- 其他公司和平博偏差大 → 大概率是其他公司在做盘/诱导
- 偏差方向就是潜在价值方向

**第五层：多家公司一致性**

- 全部一致 → 市场共识，信号可靠
- 多数一致、1家反向 → 关注反向公司是否有特殊信息渠道
- 2家以上不一致 → **重大分歧，在第8步重点分析**

#### 第 4 步：亚盘赔率分析

数据来源：步骤 1.5 抓取的各公司亚盘变化。

参考公司：

- 默认：澳彩、皇冠、365、易胜博、**平博**
- 亚洲联赛：可选加入香港马会

**数据展示**（用表格）：

```
| 公司   | 初盘盘口 | 初盘主水 | 初盘客水 | 即时盘口 | 即时主水 | 即时客水 | 判断       |
| ------ | -------- | -------- | -------- | -------- | -------- | -------- | ---------- |
| 澳彩   | {盘口}   | x.xx     | x.xx     | {盘口}   | x.xx     | x.xx     | {正常/异常} |
| ...    | ...      | ...      | ...      | ...      | ...      | ...      | ...        |
```

**分析思路（教学级排查流程）**：

**第一层：初盘合理性判断** — 结合排名、实力差、主客场，该让多少？

- **让大（强让）**：盘口超过实力差应给的值
  - 可能是**诱上盘**：故意开高引诱散户跟上盘，实际走下盘
  - 判断方法：强让+上盘高水 = 典型诱上盘；强让+上盘低水 = 真看好上盘
- **让少（浅让）**：盘口低于实力差应有的值
  - 先排查原因：伤停严重？交锋往绩不好？一周多赛轮换？战意不足？
  - 能解释 → 正常让少，机构合理反映利空
  - **解释不了 → 机构掌握场外信息，高度警惕，暗盘信号**
- **正常**：盘口与实力差匹配，合理开盘

**第二层：变动排查** — **只要盘口或水位有变化，都必须给出原因判断**

每一次变动只有三种可能，逐一排查：

1. **平衡筹码（被动调整）**：某一方投注量过大，机构调整水位/盘口来平衡两边资金
   - 特征：变动幅度小（水位变动 0.02-0.05），方向与大众情绪一致
   - 例：热门主队吸引大量投注 → 主水小幅升高 → 正常平衡，不改变判断
2. **机构主动调整（有信息驱动）**：机构基于掌握的内部信息主动改变盘口/水位
   - 特征：变动幅度大（水位变动 >0.08 或盘口直接升降），与大众情绪相反
   - 例：主队热门但机构反而降盘 → 机构掌握利空（临场伤停？内部消息？）
   - **这是最有价值的信号，必须重点关注**
3. **诱盘操作**：机构刻意引导散户往一个方向下注，实际看好相反方向
   - 特征：盘口方向与水位方向矛盾（如升盘但上盘水位反降）
   - 例：升半球但主水从高水降到低水 → 升盘引诱关注上盘，降水又让散户觉得上盘有价值 → 实际看好下盘

**排查清单（每次变动都要过一遍）**：

```
变动内容：{盘口变了 / 水位变了 / 都变了}
变动幅度：{小幅(≤0.05) / 中幅(0.05-0.10) / 大幅(>0.10) / 盘口升降}
变动时间：{早期调整 / 中段调整 / 临场突变}
与大众情绪：{同向(可能平衡筹码) / 反向(可能机构主动)}
盘口与水位是否矛盾：{一致(正常) / 矛盾(疑似诱盘)}
能否用基本面解释：{能(伤停/交锋/赛程) / 不能(暗盘信号)}
→ 结论：{平衡筹码 / 机构主动调整 / 诱盘操作 / 暗盘信号}
```

**第三层：水位信号解读**

- 主水升高 = 阻上盘（不看好主队让球）
- 主水降低 = 放上盘（看好主队让球，或诱下盘）
- 两头低水 = 机构利润空间大，对结果有较强把握
- 两头高水 = 机构在犹豫，盘口可能还会变，暂不下结论

**第四层：RLM 检测（同欧指）**

- 大众倾向买上盘（热门队让球）→ 但盘口反而降盘或上盘水位升高 = RLM 信号
- 一旦检测到，在第 8 步标记为高权重信号

**第五层：多家公司一致性**

- 全部同盘口、同方向 → 共识强，信号可靠
- 多数一致、1家反向 → 关注反向公司理由
- **2家以上不一致 → 在第8步重点分析原因**

#### 第 5 步：进球数赔率分析

数据来源：步骤 1.5 抓取的各公司大小球变化。

参考公司：

- 默认：澳彩、皇冠、365、易胜博、**平博**
- 亚洲联赛：可选加入香港马会

**数据展示**（用表格）：

```
| 公司   | 初盘盘口 | 初盘大球 | 初盘小球 | 即时盘口 | 即时大球 | 即时小球 | 判断       |
| ------ | -------- | -------- | -------- | -------- | -------- | -------- | ---------- |
| 澳彩   | {盘口}   | x.xx     | x.xx     | {盘口}   | x.xx     | x.xx     | {看大/看小} |
| ...    | ...      | ...      | ...      | ...      | ...      | ...      | ...        |
```

**分析思路（教学级排查流程）**：

**第一层：初盘合理性判断** — 主队场均进球 + 客队场均进球 ≈ 预期总进球，对应合理盘口。

- **盘口偏高（开大了）**：初盘高于场均预期
  - 可能是**诱大球**：引诱散户买大球，实际看小球
  - 判断方法：高盘口+大球高水 = 诱大球信号；高盘口+大球低水 = 真看好大球
- **盘口偏低（开小了）**：初盘低于场均预期
  - 可能是**诱小球**，也可能真看好小球（防守型打法、雨战、伤停核心射手等）
  - 需排查原因：能解释则正常，不能解释则可能诱小球
- **正常**：盘口与场均预期基本匹配

**第二层：变动排查** — 与亚盘相同，**只要盘口或水位有变化，都必须判断原因**

同样逐一排查三种可能：

1. **平衡筹码（被动调整）**：某一方投注量大，小幅调水位平衡
   - 特征：水位变动小（≤0.05），方向与热门方一致
2. **机构主动调整**：基于信息主动改变判断
   - 特征：变动幅度大或盘口直接升降
   - 例：从 2.5球 降到 2/2.5球 = 机构主动调低进球预期，排查是否有伤停或战术变化
3. **诱盘操作**：盘口方向与水位方向矛盾
   - 例：升盘（盘口变大）但大球水位反升 → 升盘引诱关注大球，实际看小球

**排查清单（同亚盘）**：

```
变动内容：{盘口变了 / 水位变了 / 都变了}
变动幅度：{小幅 / 中幅 / 大幅 / 盘口升降}
能否用基本面解释：{能(射手伤停/防守型打法/天气) / 不能}
盘口与水位是否矛盾：{一致 / 矛盾(疑似诱盘)}
→ 结论：{平衡筹码 / 机构主动看大(小) / 诱大(小)球}
```

**第三层：与基本面交叉验证**

- 两队都是攻强守弱 + 盘口却偏低 → 矛盾，可能诱小球
- 两队都是防守型 + 盘口偏高 → 矛盾，可能诱大球
- 核心射手伤停 + 盘口降低 → 合理，机构正确反映信息
- 无明显原因但盘口大幅变动 → 暗盘信号

**第四层：RLM 检测 + 平博验证**

- RLM：大众倾向买大球（进攻型强队比赛）→ 但盘口降低或大球水位升高 = RLM 信号
- 平博验证：平博(47)大小球方向 = 尖锐资金对进球数的判断，与其他公司不一致时要警惕

**第五层：多家公司一致性**

- 全部同盘口同方向 → 信号强
- 有分歧 → 在第8步重点分析原因

#### 第 6 步：赔率合理性评估

**三种赔率交叉验证**：

- 欧指、亚盘、大小球三者方向是否自洽？
  - 例：欧指看好主胜，亚盘主队让球合理，大小球偏大 → 三者一致，信号强
  - 例：欧指看好主胜，但亚盘让球偏少 → 存在矛盾，需警惕
- 赔率与基本面是否匹配？
  - 基本面利好主队但赔率不支持 → 可能机构掌握了额外信息（伤停、战意等）
  - 基本面一般但赔率明显倾向某方 → 可能有「暗盘」信号
- 是否存在「诱盘」迹象？（参考第 3-5 步的排查结论）

**价值评估（辅助参考）**：

借鉴 Tony Bloom / Starlizard 的核心方法——先建自己的概率判断，再与赔率隐含概率对比，寻找定价偏差。

```
1. 基于前5步分析，估算自己的概率判断：
   - 主胜概率：约 X%
   - 平局概率：约 X%
   - 客胜概率：约 X%

2. 计算赔率隐含概率（去除抽水）：
   - 隐含主胜 = 1/主胜赔率，隐含平 = 1/平赔，隐含客胜 = 1/客胜赔率
   - 总和 > 100% 的部分是博彩公司抽水，按比例扣除得到公允概率

3. 对比差异：
   - 自己判断 > 隐含概率 → 该方向存在价值（正期望值）
   - 自己判断 < 隐含概率 → 该方向无价值
   - 差异越大，价值越高
```

注意：此概率估算基于有限数据，精度不如专业量化模型，**仅作为辅助参考验证推荐方向是否有正期望值**，不作为核心决策依据。如果价值评估与盘赔分析方向一致，信心加分；方向相反，则需在第8步分析原因。

**平博锚点校准**：将自己的概率估算与平博(47)欧指隐含概率对比。如果自己的判断与平博接近，说明分析靠谱；如果偏差大，需要反思是自己遗漏了信息，还是确实发现了平博尚未反映的价值。

输出格式：明确指出三种赔率之间的一致性/矛盾点 + 价值评估结论（哪个方向有价值/无价值）+ 与平博隐含概率的偏差分析。

#### 第 7 步：赔率变动解读

分析各公司赔率随时间的变化趋势：

- 初盘 → 中间变动 → 即时盘的时间线
- 哪些公司率先变动？（率先变动的公司往往掌握更多信息）
- 临场大幅变动（封盘前1-2小时的突变）含义
- 各公司变动方向是否一致：
  - **全部一致**：市场共识强，跟随即可
  - **多数一致、少数反向**：关注反向公司的原因，但可跟随多数
  - **方向分散**：市场无共识，风险高，谨慎推荐

输出格式：描述关键变动节点和各公司动向。

#### 第 8 步：关键点 / 疑点 / 矛盾分析

综合前 7 步，提炼出本场比赛的核心判断要素：

- **关键点**：最能决定比赛走向的 1-2 个因素（如「主队中锋伤停+客队6连胜」）
- **疑点**：分析中存在不确定的地方（如「客队3天前有杯赛，轮换程度未知」）
- **矛盾**：各维度或各公司之间的冲突信号（如「基本面看好主队，但亚盘让球偏低」）

这一步是整个分析最核心的思考环节。

#### 第 9 步：风险提示

明确列出可能推翻判断的因素：

- 伤停变化（临场可能有新增伤停）
- 天气因素（暴雨/极寒等）
- 心理因素（德比战/保级战情绪波动）
- 赔率临场异动风险
- 其他不可预见因素

### 步骤 3 续：第 10 步 — 比分概率建模（泊松分布）

**这是龙王最核心的需求——比分预测。** 基于前 9 步分析的结论，用泊松分布计算比分概率排序。

#### 计算方法

**1. 确定双方预期进球数（λ）**

从分析页提取得失球数据，计算：

```
主队预期进球(λ1) = 主队场均进球 × 客队场均失球 / 联赛场均进球
客队预期进球(λ2) = 客队场均进球 × 主队场均失球 / 联赛场均进球
```

如果联赛场均进球数据不可用，简化为：

```
λ1 = (主队场均进球 + 客队场均失球) / 2
λ2 = (客队场均进球 + 主队场均失球) / 2
```

**2. 结合分析结论修正 λ 值**

泊松模型的 λ 需要结合前 9 步分析做主观修正：

- 核心射手伤停 → 降低该队 λ（降幅 0.1-0.3，视球员重要性）
- 防守核心伤停 → 提高对手 λ
- 主场优势 → 主队 λ 上浮 5-10%（如果主场龙属性强则更多）
- 赛程陷阱/轮换 → 降低该队 λ
- 教练风格（防守型遇进攻型）→ 适当调整
- 大小球盘赔倾向 → 如果盘赔指向小球，整体下调 λ

**⚠️ λ 合理性检查**：所有修正完成后，检查最终 λ 值是否在合理范围内：

- **下限 0.4**：低于 0.4 意味着该队约 67% 概率进 0 球，除非是极端防守局面（如保级队客场 vs 豪门），否则不合理
- **上限 3.5**：高于 3.5 意味着预期场均 3.5 球，即使是最强进攻队也极为罕见
- 如果修正后 λ 超出 [0.4, 3.5] 范围，回退到边界值，并在分析中标注「λ 修正幅度异常，已截断至边界值，比分概率仅供参考」

**3. 计算各比分概率**

用泊松公式计算 0:0 到 7:7 的每个比分概率（覆盖至 7 球，确保高 λ 时概率总和 > 97%）：

```
P(主队进k球) = e^(-λ1) × λ1^k / k!
P(客队进j球) = e^(-λ2) × λ2^j / j!
P(比分k:j) = P(主队进k球) × P(客队进j球)
```

**⚠️ 计算精度要求**：涉及 64 个比分组合的浮点运算，**必须使用代码执行**，不要心算或手动推导。

**Dixon-Coles 修正**：基础泊松假设主客队进球独立，但实际上低比分（0:0, 1:0, 0:1, 1:1）的出现频率高于独立预测。用修正因子 ρ ≈ -0.13（欧洲顶级联赛经验值）调整这四个比分的概率，提升比分预测准确度。

**执行方式**：使用 Shell 工具执行 Python 命令，不依赖浏览器标签页状态，独立可靠。**执行前务必将模板中的 `LAMBDA1` 和 `LAMBDA2` 替换为实际数字**（如 1.35、0.92），否则会报 `NameError`。项目内 `scripts/verify_poisson.py` 可验证该公式与输出。

```bash
python3 -c "
import math
L1, L2 = LAMBDA1, LAMBDA2  # 替换为实际 λ 值，如 1.35, 0.92
rho = -0.13  # Dixon-Coles 修正因子（低比分相关性）
def p(l,k): return math.exp(-l)*l**k/math.factorial(k)
scores = []
for i in range(8):
    for j in range(8):
        prob = p(L1,i) * p(L2,j)
        if i==0 and j==0: prob *= 1 - L1*L2*rho
        elif i==1 and j==0: prob *= 1 + L2*rho
        elif i==0 and j==1: prob *= 1 + L1*rho
        elif i==1 and j==1: prob *= 1 - rho
        scores.append((f'{i}:{j}', max(prob, 0)))
total = sum(pr for _,pr in scores)
scores = [(s, pr/total) for s,pr in scores]
scores.sort(key=lambda x: -x[1])
for s,pr in scores: print(f'{s}: {pr*100:.2f}%')
"
```

**4. 输出比分推荐（3 个，最多 4 个）**

按概率从高到低排列，只给最优的 3 个比分（信心高且比分间概率差距不大时可给第 4 个）：

```
| 排名 | 比分 | 概率   | 胜平负 | 让球({盘口}) | 大小({盘口}) |
| ---- | ---- | ------ | ------ | ------------ | ------------ |
| 1    | 1:0  | 12.3%  | 主胜   | 上盘 ✅      | 小球 ✅      |
| 2    | 2:1  | 10.8%  | 主胜   | 下盘 ❌      | 大球 ✅      |
| 3    | 1:1  | 10.2%  | 平局   | 下盘 ❌      | 小球 ✅      |
```

内部仍计算完整概率分布（用于价值偏差分析和复盘追踪），但展示给龙王的只有 Top 3-4。

**5. 与竞彩比分赔率交叉验证（价值发现）**

这是比分投注的核心——找出竞彩定价偏差：

- 竞彩比分赔率隐含概率 = 1 / 赔率（需去除抽水）
- 泊松概率 > 竞彩隐含概率 → **该比分有正期望值（高价值）**
- 泊松概率 < 竞彩隐含概率 → 该比分无价值
- 差距越大，价值越高

```
| 比分 | 泊松概率 | 竞彩赔率 | 竞彩隐含概率 | 价值偏差 | 标记       |
| ---- | -------- | -------- | ------------ | -------- | ---------- |
| 1:0  | 12.3%    | 7.50     | 11.2%        | +1.1%    | 💰 有价值  |
| 2:1  | 10.8%    | 8.00     | 10.5%        | +0.3%    | ≈ 持平     |
| 0:0  | 8.1%     | 9.00     | 9.3%         | -1.2%    | ❌ 无价值  |
```

**数据来源**：titan007 的比赛分析页包含比分相关数据。在步骤 1 抓取分析页时，留意页面中的竞彩比分赔率/赔付数据模块，一并提取。如果页面上确实没有比分赔率数据，则跳过价值偏差计算，只输出泊松概率排序。

基于 9 步分析 + 比分概率建模，汇总给出最终结论：

**信心度判断规则**：

- 基本面 + 盘赔 + 赔率合理性三者一致，且无重大矛盾 → **高**
- 多数分析方向一致，有小分歧但可解释 → **中**
- 存在明显矛盾或公司方向分散 → **低**，考虑不推荐

**置信区间推导**：

- 基于泊松模型概率和盘赔隐含概率，给出一个范围而非单一数字
- 范围宽度反映不确定性：信心高 → 窄区间（如 55-65%），信心低 → 宽区间（如 40-60%）

**串关适合度规则**：

- 🔒 **做胆推荐**：信心度高 + 赛果方向非常明确 + 赔率合理 + 无 RLM 逆向信号
- ✅ **可串**：信心度中 + 方向较明确 + 盘赔无重大分歧
- ⚠️ **慎串**：有矛盾信号或不确定性大，单买可以但串关会放大风险

**比分预测推导**（泊松 + 分析修正）：

1. 以泊松模型计算的概率排序为基础
2. 结合大小球倾向和盘口，确认进球区间是否与泊松一致
3. 结合胜平负倾向，在高概率比分中优先选择方向一致的
4. 考虑交锋历史中的常见比分模式（如某对交锋常出 1:1）
5. 标注哪些比分有竞彩价值偏差（泊松概率 > 竞彩隐含概率）
6. 输出 Top 3 比分（最多 4 个）供龙王选择串关组合

### 步骤 5：写入记忆

将分析结果追加写入 `memory/{今天日期}.md`。

## 异常处理

### 单页面级异常

- **详情页加载失败**：跳过浏览器数据，用已有信息（初筛阶段获取的）做简化分析，标注「数据不完整」
- **某个步骤完全无数据**：标注「本步骤无数据，跳过」，用剩余步骤综合判断
- **赔率数据全灭**（vip.titan007.com 不可用，所有公司赔率页面均失败）：
  - 尝试从分析页顶部的「即时走势（让/大/欧）」标签获取汇总盘口作为替代
  - 如果汇总盘口也无法获取 → 该场降级为「仅基本面分析」，信心度自动设为「低」，串关适合度设为「⚠️慎串」
  - 在推送中明确标注「⚠️ 赔率数据不可用，本场仅基于基本面分析，建议谨慎」
  - 不影响其他场次的分析（可能只是某场的赔率页面有问题）

### 浏览器级异常

- **浏览器崩溃/无响应**：
  1. 重启浏览器（`browser restart` 或等待 OpenClaw 自动恢复）
  2. 从当日 memory 检查 `## 深度分析` 中已分析完的场次（有完整综合评估的）
  3. 跳过已完成的场次，从下一场继续
  4. 已推送给龙王的分析不受影响

- **疑似限流/封 IP**（连续 3 个以上页面返回空数据或超时）：
  1. 暂停所有浏览器操作，等待 60 秒
  2. 尝试访问一个简单页面（如赛程列表首页）验证连通性
  3. 恢复正常 → 继续分析，后续每页间隔增加到 3 秒
  4. 仍然失败 → 对剩余场次改用分析页「即时走势」汇总盘口做简化赔率分析，标注「赔率详情不可用，使用汇总数据」

### 部分完成场景

- **已推送 N 场后中断**：已推送的场次不受影响。恢复后只处理未完成的场次。最终汇总包含所有已完成场次（含中断前的）
- **某场分析中途失败**（如基本面抓完但赔率没抓完）：用已有数据完成分析，缺失部分标注「数据不完整」，信心度酌情下调

### 频率控制

- **分析场次 ≤ 5 场**：每页间隔 1-2 秒
- **分析场次 > 5 场**：每页间隔 2-3 秒，每完成一场后额外暂停 3 秒

返回每场比赛的**完整分析过程 + 综合评估结论**，供 recommendation 推送给龙王。分析过程是推送消息的核心内容，不能省略。
