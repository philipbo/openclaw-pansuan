## 泊松计算流程（第 4 步）

### 1. λ 值计算（预期进球）

**标准公式**（优先使用，需要联赛场均数据）：
```
主队 λ1 = 主队场均进球 × 客队场均失球 / 联赛场均进球
客队 λ2 = 客队场均进球 × 主队场均失球 / 联赛场均进球
```

**简化公式**（联赛场均数据不可用时）：
```
λ1 = (主队场均进球 + 客队场均失球) / 2
λ2 = (客队场均进球 + 主队场均失球) / 2
```

**数据来源**：步骤 1 从 titan007 分析页抓取的场均进/失球数据

---

### 2. λ 值修正（基于基本面和伤停）

**修正因素**（可叠加，但同一因素不重复修正）：

| 因素 | 修正方式 | 示例 |
|------|---------|------|
| 核心射手伤停 | 该队 λ - 0.1～0.3 | 主力前锋缺阵 → λ - 0.2 |
| 防守核心伤停 | 对手 λ + 0.1～0.2 | 主力中卫缺阵 → 对手 λ + 0.15 |
| 主场优势 | 主队 λ × 1.05～1.10 | 主场龙 → λ × 1.10 |
| 客场疲软 | 客队 λ × 0.90～0.95 | 客场虫 → λ × 0.90 |
| 赛程陷阱/轮换 | 该队 λ × 0.90～0.95 | 3 天 2 赛 → λ × 0.92 |
| 大小球盘赔倾向 | 双方 λ 各 × 0.95～1.05 | 盘赔看小 → λ × 0.95 |

**λ 合理性检查**：
- 下限 **0.4**（低于此值约 67% 概率进 0 球）
- 上限 **3.5**（高于此值过于极端）
- 超出范围则截断至边界值

---

### 3. 泊松概率计算

**基础公式**：
```
P(主队进 k 球) = e^(-λ1) × λ1^k / k!
P(客队进 j 球) = e^(-λ2) × λ2^j / j!
P(比分 k:j) = P(主队进 k 球) × P(客队进 j 球)
```

**Dixon-Coles 修正**（ρ = -0.13 默认）：
- 0:0 → 概率 × (1 - λ1×λ2×ρ)
- 1:0 → 概率 × (1 + λ2×ρ)
- 0:1 → 概率 × (1 + λ1×ρ)
- 1:1 → 概率 × (1 - ρ)

**ρ 取值指引**：
| 联赛类型 | ρ 值 |
|---------|-----|
| 欧洲五大联赛、欧冠、欧联 | -0.13 |
| 高进球联赛（荷甲、巴甲等场均>2.8） | -0.08 |
| 低进球/防守联赛（日职联、土超等场均<2.2） | -0.15 |

---

### 4. 计算执行方式

**必须用 Python 代码执行**（64 个比分组合的浮点运算）：

```python
import math
L1, L2 = 1.60, 0.72  # 实际 λ 值
rho = -0.13  # 默认值

def p(l,k): return math.exp(-l)*l**k/math.factorial(k)

scores = []
for i in range(8):
    for j in range(8):
        prob = p(L1,i) * p(L2,j)
        # Dixon-Coles 修正
        if i==0 and j==0: prob *= 1 - L1*L2*rho
        elif i==1 and j==0: prob *= 1 + L2*rho
        elif i==0 and j==1: prob *= 1 + L1*rho
        elif i==1 and j==1: prob *= 1 - rho
        scores.append((f'{i}:{j}', max(prob, 0)))

# 归一化 + 排序
total = sum(pr for _,pr in scores)
scores = [(s, pr/total) for s,pr in scores]
scores.sort(key=lambda x: -x[1])

# 输出 Top 6
for s,pr in scores[:6]: print(f'{s}: {pr*100:.2f}%')
```
