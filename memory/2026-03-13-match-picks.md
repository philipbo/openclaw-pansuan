## 🔧 Gateway 是什么？

Gateway 是 OpenClaw 的核心服务，负责：
- 工具路由（包括 browser、messaging 等）

## 🐛 为什么会超时？

可能的原因：
1. **Browser Relay 断开** — Chrome 扩展连接丢失
2. **浏览器进程卡死** — 之前的页面没关闭干净
3. **网络/端口问题** — 本地服务端口被占用

```bash
openclaw gateway restart
```

**方法 3：先停后启**
```bash
openclaw gateway stop
openclaw gateway start
```
