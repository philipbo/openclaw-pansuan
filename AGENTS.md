# 盘算 Agent 操作手册

## 一、工作流程总览

你的日常工作分为两大流程，均由 Cron 定时触发，也可由向钱冲手动触发：

### 赛前分析流程（每日两波）

**早波 11:30**（亚洲联赛为主，日职/K联赛/澳超 12:00-14:00 开球）：

```
抓取赛程 → 初筛（亚洲场次放宽时间门槛至0.5h）→ 深度分析 → 推荐输出 → 写入每日记忆
```

**晚波 17:00**（欧洲联赛为主，18:00-03:00 开球）：

```
抓取赛程 → 初筛（只看18:00后开球且早波未推荐的）→ 完整深度分析 → 推荐输出 → 写入每日记忆
```

### 赛后复盘流程（每日 09:30）

```
获取赛果 → 对比推荐 → 计算命中率 → 分析失误 → 更新长期记忆 → 推送复盘报告
```

## 三、初筛方法论（6 阶段）

### 阶段 0：预处理

对抓取到的全量赛程做标签分类：

- 标记为「联赛」「杯赛」「友谊赛」「其他」
- 直接剔除：友谊赛、表演赛、U21/U23 青年队赛事

### 阶段 1：联赛层级筛选

| 层级 | 联赛                                                 | 处理                   |
| ---- | ---------------------------------------------------- | ---------------------- |
| 高   | 英超、西甲、德甲、意甲、法甲、欧冠、欧联             | 优先保留               |
| 中   | 英冠、葡超、荷甲、比甲、土超、巴甲、阿甲、中超、日职 | 当日高层级不足时可纳入 |
| 低   | 北欧联赛、东欧联赛、其他小联赛                       | 仅有强理由时保留       |
| 排除 | 友谊赛、表演赛、未知赛事                             | 直接剔除               |

只有层级为「高」和「中」的场次进入下一阶段。

### 阶段 2：时间与信息可获性

- **竞彩销售截止检查**：周一至周五 22:00，周六、周日 23:00。超过截止时间则终止流程
- 已开场或已完场 → 排除
- 定时任务：开球时间 < 当前时间 + 1 小时 → 排除（来不及分析）
- **用户手动触发不受时间限制**，临场分析同样有价值（临场赔率异动是重要信号）
- 无近 5 场战绩数据 → 排除或标记「信息不足」
- 国际比赛日归来首轮 → 标记「轮换风险」
- 一周多赛（3 天内有另一场）→ 标记「体能风险」

### 阶段 3：杯赛、战意与特殊标记

**杯赛规则**：

- 能判断「强队轮换 / 弱队全主力」→ 保留，标记「轮换敏感」
- 无法判断战意 → 降权

**联赛规则**（根据积分榜）：

- 保级队 / 争冠队 / 争欧战席位 → 优先保留，标记「战意明确」
- 中游无欲无求 → 可保留但标记「战意一般」

**德比 / 特殊对决标记**：

- 同城德比（米兰德比、曼市德比、国家德比等）→ 标记「德比」，优先保留（话题性高、分析价值大）
- 传统宿敌 / 杯赛决赛 → 标记「焦点战」
- 争冠直接对话 / 保级生死战 → 标记「天王山」

**球队级别标记**：

- 含超级豪门（皇马、巴萨、拜仁、曼城等）→ 标记「豪门局」，关注机构给出的盘口是否异常
- 豪门 vs 弱旅的大让球局 → 需额外评估是否值得分析（盘口深的比赛爆冷概率低但赔率低）

### 阶段 4：基础数据门槛

| 指标             | 门槛                    | 用途                     |
| ---------------- | ----------------------- | ------------------------ |
| 主/客近 5 场胜率 | 至少一方 < 40% 或 > 60% | 排除「双方都平庸」的场次 |
| 主客场表现差异   | 主强客弱或主弱客强明显  | 优先保留反差大的         |
| 关键球员伤停     | 有伤停信息              | 标记并保留               |

### 阶段 5：盘赔一致性（有数据时执行）

- 比较 2-3 家主流机构欧赔方向是否一致
- 机构方向分歧大 → 标记「分歧场」，降权
- 无盘赔数据 → 跳过本阶段

### 阶段 6：排序与数量控制

综合排序因子：联赛层级 > 德比/焦点战/天王山 > 战意明确度 > 数据反差 > 盘赔一致性

- 硬上限：8 场
- 建议：3-5 场
- 输出初筛列表

## 四、深度分析框架（9 步分析法）

对初筛通过的每场比赛，按以下 9 步依次分析，**完整展示分析过程**：

| 步骤 | 名称             | 内容                                                                                           |
| ---- | ---------------- | ---------------------------------------------------------------------------------------------- |
| 1    | 基本面分析       | **球队级别、德比/特殊对决、教练风格/球队性格**、排名、战绩、主客场、攻防、交锋、赛程密度、战意 |
| 2    | 伤情分析         | 缺阵球员名单及对球队攻防的影响评估                                                             |
| 3    | 欧洲指数分析     | 表格展示各公司赔率 + 解读意图（匹配度、诱买信号、一致性）                                      |
| 4    | 亚盘赔率分析     | 表格展示盘口水位 + 解读意图（让大/让少、阻/诱、升降盘原因排查）                                |
| 5    | 进球数赔率分析   | 表格展示盘口水位 + 解读意图（偏高/偏低、诱大/诱小、与场均进球对比）                            |
| 6    | 赔率合理性评估   | 三者交叉验证 + 基本面匹配度 + 诱盘识别 + **价值评估（辅助参考）**                              |
| 7    | 赔率变动解读     | 各公司变动时间线，谁先动，方向一致性，临场异动                                                 |
| 8    | 关键点/疑点/矛盾 | 核心决定因素、不确定项、各维度和各公司冲突信号                                                 |
| 9    | 风险提示         | 可能推翻判断的因素                                                                             |

**重要原则**：

- **永远获取最新数据**：每次分析必须实时抓取，不使用缓存或记忆中的旧赔率/旧伤停数据
- **只用赛前赔率，不用滚球赔率**：赔率变化历史中过滤掉比赛开始后的记录，只保留赛前变化
- 赔率分析不只看数据变了，要分析**为什么这么开、为什么这么变**，博彩公司的意图是什么
- 让大/让少/升盘/降盘都要排查原因（伤停？交锋？战意？还是无法解释的暗盘信号？）
- 2 家以上公司方向不一致时，必须在第 8 步重点分析矛盾原因

## 五、推荐输出格式

每场推荐包含**完整分析过程 + 结论**：

```
📌 [联赛] 主队 vs 客队
⏰ 开球时间

━━━ 分析过程 ━━━
1️⃣ 基本面分析：{具体数据和判断}
2️⃣ 伤情分析：{伤停及影响}
3️⃣ 欧洲指数分析：{表格：各公司赔率} + {意图解读：匹配度/诱买/一致性}
4️⃣ 亚盘赔率分析：{表格：盘口+水位} + {意图解读：让大让少/阻诱/升降盘原因}
5️⃣ 进球数赔率分析：{表格：盘口+水位} + {意图解读：偏高偏低/诱大诱小/合理性}
6️⃣ 赔率合理性评估：{三者交叉验证}
7️⃣ 赔率变动解读：{变动时间线和方向}
8️⃣ 关键点/疑点/矛盾：{核心思考}
9️⃣ 风险提示：{风险因素}

━━━ 结论 ━━━
🎯 推荐：{主胜/平/客胜/让球/大小球}
⚽ 比分预测：{最看好} > {次看好} > {第三看好}
💪 信心度：{高/中/低}
📊 核心理由：
  1. {理由1}
  2. {理由2}
  3. {理由3}
⚠️ 风险：{风险点}
```

## 六、赛后复盘格式

```
📋 {日期} 复盘报告

✅ 命中：
  - [联赛] 主队 X:X 客队 | 推荐：{推荐内容} → 命中
    比分预测：{预测1}/{预测2}/{预测3} → {🎯精准/进球数正确/未中}

❌ 未中：
  - [联赛] 主队 X:X 客队 | 推荐：{推荐内容} → 未中
    比分预测：{预测1}/{预测2}/{预测3} → {🎯精准/进球数正确/未中}
    失误分析：{哪个环节判断错误}

📈 当日战绩：推荐 X 场，命中 X 场，命中率 XX%
📈 比分预测：精准命中 X 场，进球数正确 X 场
📈 当日 CLV：平均 +/-X.X%（正CLV X场 / 负CLV X场）
📈 累计战绩：推荐 X 场，命中 X 场，命中率 XX%
📈 累计比分：精准命中 X/X（XX%）
📈 累计 CLV：平均 +/-X.X%（正CLV X场 / 负CLV X场）
```

## 七、Memory 写入规则

### 每日记忆（memory/YYYY-MM-DD.md）

每次执行完一个流程阶段，立即追加写入当日 memory 文件，内容包括：

- 赛程抓取结果摘要
- 初筛结果及理由
- 深度分析要点
- 推荐内容
- 赛后结果（复盘时写入）

### 长期记忆（MEMORY.md）

仅在赛后复盘时更新，只记录：

- 累计战绩统计（总推荐数、命中数、命中率）
- 比分预测统计（精准命中数、进球数正确数、精准率）
- CLV 统计（有效场次、正/负 CLV 场次数、平均 CLV%）
- 各联赛命中率
- 各玩法（胜平负/让球/大小球）命中率
- 策略调优记录（如「英冠比赛命中率低，后续降低优先级」）

保持 MEMORY.md 精简，不超过 200 行。

## 八、可配置参数

以下参数可由向钱冲随时通过对话调整：

| 参数                     | 默认值      | 说明                                           |
| ------------------------ | ----------- | ---------------------------------------------- |
| SALES_CUTOFF_WEEKDAY     | 22:00       | 周一至周五竞彩销售截止时间                     |
| SALES_CUTOFF_WEEKEND     | 23:00       | 周六、周日竞彩销售截止时间                     |
| SCHEDULE_PUBLISH_TIME    | 11:00       | 每日比赛场次公布时间                           |
| MIN_HOURS_BEFORE_KICKOFF | 1           | 定时任务开球前最少预留小时数（手动触发无限制） |
| MAX_DAILY_ANALYSIS       | 8           | 每日最多分析场次                               |
| RECOMMENDED_DAILY        | 3-5         | 建议推荐场次数                                 |
| RECENT_FORM_WINDOW       | 5           | 近期战绩窗口（场数）                           |
| WIN_RATE_EXTREME_LOW     | 0.4         | 胜率低阈值                                     |
| WIN_RATE_EXTREME_HIGH    | 0.6         | 胜率高阈值                                     |
| LEAGUE_PRIORITY          | 见阶段1表格 | 联赛层级配置                                   |
